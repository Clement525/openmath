Uniformly Continuous Function
We say that \( f : S \subseteq \mathbb{ R } ^ n \to \mathbb{ R } ^ m \) is uniformly continuous if for every \( \epsilon \in \mathbb{ R } ^ + \) there exists some \( r \in \mathbb{ R } ^ + \) such that for every \( x, a \in S \) we have \[ \lVert x - a \rVert \lt r \implies \lVert f \left( x \right) - f \left( a \right) \rVert \lt \epsilon \]

Going to Infinity Implies not Uniformly Continuous
Suppose that \( f \) is continuous on \( \left( 0, 1 \right) \) and that \( \lim _ { x \to 0 ^ + } f \left( x \right) = \infty \) show that \( f \) is not uniformly continuous

Suppose for the sake of contradiction that \( f \) is uniformly continuous and take \( \epsilon = 1 \) therefore we obtain some \( r \in \mathbb{ R } ^ + \) such that for any \( x, a \in S \) if \( \left\lvert x - a \right\rvert \lt r \) implies that \( \left\lvert f \left( x \right) - f \left( a \right) \right\rvert \lt \epsilon = 1 \).

Note that this implies that for any \( a, b \in \left( 0, r \right) \) we have \( \left\lvert f \left( a \right) - f \left( b \right) \right\rvert \lt 1 \) clearly this will lead to nonsense as it blows up around \( 0 \) and so we should be able to find two points whose vertical distance is greater or equal to \( 1 \).

To do this let \( x \in \left( 0, r \right) \), since we know that \( \lim _ { x \to 0 ^ + } f \left( x \right) = \infty \) therefore by taking \( \delta = r, M = f \left( x \right) + 1 \) we obtain some \( a \in \left( 0, r \right) \) such that \( f \left( a \right) \gt M = f \left( x \right) + 1 \) therefore we have that \( f \left( a \right) - f \left( x \right) \gt 1 \) which implies that \[ 1 \gt \left\lvert f \left( a \right) - f \left( x \right) \right\rvert = \left\lvert f \left( x \right) - f \left( a \right) \right\rvert \] which is a contradiction, so that \( f \) is uniformly continuous.
