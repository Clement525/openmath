Sign of the Derivative Tells us about Monotonicity
Let \( f \) be differentiable on \( \left[ a, b \right] \) then if for all \( x \in \left( a, b \right) \) we have:
  • \( f ^ \prime \left( x \right) = 0 \) then \( f \) is constant
  • \( f ^ \prime \left( x \right) \gt 0 \) then \( f \) is strictly increasing
  • \( f ^ \prime \left( x \right) \lt 0 \) then \( f \) is strictly decreasing
  • \( f ^ \prime \left( x \right) \ge 0 \) then \( f \) is increasing
  • \( f ^ \prime \left( x \right) \le 0 \) then \( f \) is decreasing
Matching Derivatives Implies Differing by a Constant
Suppose that \( f, g : \mathbb{ R } \to \mathbb{ R } \) and that \( f ^ \prime \left( x \right) = g ^ \prime \left( x \right) \) for all \( x \in \left( a, b \right) \) then show that \[ g \left( x \right) = f \left( x \right) + c \] for some \( c \in \mathbb{ R } \)
Observe that we have \( f ^ \prime \left( x \right) - g ^ \prime \left( x \right) = 0 \) and therefore \( f - g \) is constant so that there is some \( c \in \mathbb{ R } \) such that \( f \left( x \right) - g \left( x \right) = c \) so we have \[ f \left( x \right) = g \left( x \right) + c \] as needed.
Non-Zero Derivative Implies Locally One-to-One
Suppose that \( f: \left( a, b \right) \to \mathbb{ R } \) has continuous derivative on \( \left( a, b \right) \). If \( f ^ \prime \left( x _ 0 \right) \neq 0 \) prove that there is an interval \( \left( a, c \right) \) containing \( x _ 0 \) such that \( f \) is one to one on \( \left( a, c \right) \)
Without loss of generality assume that \( f \left( x _ 0 \right) \gt 0 \) and recall that \( \mathbb{ R } ^ + \) is an open set, therefore since \( f ^ \prime \) is continuous then we know that \( V := \left( f ^ \prime \right) ^ { -1 } \left( \mathbb{ R } ^ + \right) \) is open and since \( f ^ \prime \left( x _ 0 \right) \gt 0 \) then \( x _ 0 \in V \) therefore there exists some \( x _ 0 \in \left( c, d \right) \subseteq V = \left( f ^ \prime \right) ^ { -1 } \left( \mathbb{ R } ^ + \right) \) therefore \( f ^ \prime \left( \left( c, d \right) \right) \subseteq \mathbb{ R } ^ + \) so that \( f \) is strictly increasing on \( \left( c, d \right) \) so that it is injective, as needed.

Forced Limit of the Derivative
Suppose that \( f \) is continuous on the interval \( \left[ a, b \right] \) and is differentiable at all points of \( \left( a, b \right) \) except possibly at the single point \( x _ 0 \in \left( a, b \right) \). Show that if \( \lim _ { x \to x _ 0 } f ^ \prime \left( x \right) \) exists (and denote its value by \( L \)) , show that \( f ^ \prime \left( x _ 0 \right) \) exists and \( f ^ \prime \left( x _ 0 \right) = \lim _ { x \to x _ 0 } f ^ \prime \left( x \right) \)

In order to show that \( f ^ \prime \left( x _ 0 \right) \) exists we can show that the right and left limits exist and are equal.

Consider \( \lim _ { h \to 0 ^ + } \frac{f \left( x _ 0 + h \right) - f \left( x _ 0 \right) }{h} \), for any \( h \in \mathbb{ R } ^ + \) such that \( x _ 0 + h \lt b \) we consider the interval \( \left[ x _ 0, x _ 0 + h \right] \) since \( f \) is continuous here, then there exists some \( c _ h \in \left( x _ 0, x _ 0 + h \right) \) such that \[ f ^ \prime \left( c _ h \right) = \frac{f \left( x _ 0 + h \right) - f \left( x \right) }{h} \] Note that as \( h \to 0 \) then \( c _ 0 \to x _ 0 \) therfore we have \[ L = \lim _ { c _ h \to x _ 0 } f \prime \left( c \right) = \lim _ { h \to 0 ^ + } \frac{f \left( x _ 0 + h \right) - f \left( x \right) }{h} \] we can do the same thing from the left to show that both sides of the limit are equal and both equal \( L \) as needed.
