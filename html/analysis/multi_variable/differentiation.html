directional derivative
Let `f: RR^n to R` and let `a, v in RR^n`, then the directional derivative of `f` at `a` along `v` is defined as
`grad_v f(a) := lim_(h to 0) (f(a + hv) - f(a)) / h`
Partial Derivative
Let `f: RR^n to R` then the partial derivative with respect to `x_i` is the directional derivative of `f` at `a` along `e_i`
`del/(del x_i) f(a) := grad_(e_i) f(a)`
Frozen Function
Suppose that `f: RR^n to RR` and `a in RR^n` then we can define a new function `f_(i, a): RR to RR` by `f_(i, a)(h) := f(a + he_i)`. This functin can be thought of looking the sample point `a` and then freezing in all directions except for along the `i`-th component of the input for `f`
Partial Derivative becomes frozen function derivative
`(del f) / (del x_i)(a) = d/(dh) [f_(i,a)(h)]`

differentaibility
a function `f: A to RR^m` with `A sube RR^n` is differentiable at the point `a in RR^n` if there exists a linear transformation `T: RR^n to RR^m` such that
`lim_{x to a} norm(f(x) - (f(a) + T(x - a)))/ norm(x - a) = 0`
continuous partials implies differentiability
If all the partial derivatives of a function exist in a neighborhood of a point `x_0` and are continuous at the point `x_0`, then the function is differentiable at that point `x_0`
