Sequential and Open Set Equivalences for Continuity
For a function mapping \( S \subseteq \mathbb{ R } ^ n \) into \( \mathbb{ R } ^ m \) the following are equivalent
  • \( f \) is continuous on \( S \)
  • For every convergent sequence \( \left( x _ n \right) : \mathbb{ N } _ 1 \to \mathbb{ R } ^ n \) such that \( \left( x _ n \right) \to a \in S \) then \[ \lim _ { n \to \infty } f \left( x _ n \right) = f \left( a \right) \]
  • For every open set \( U \subseteq \mathbb{ R } ^ m \), the set \( f ^ { -1 } \left( U \right) \) is open in \( S \)
