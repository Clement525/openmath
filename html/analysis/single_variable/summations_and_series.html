Harmonic Number
We define the \( n \) -th harmonic number as \[ H _ n := 1 + \frac{1 }{2} + \frac{1}{3} + \ldots + \frac{1}{n} = \sum _ { k = 1 } ^ n \frac{1}{k} \]
Closed form of a Geometric Summation
For any \( x \in \mathbb{ R } ^ { \neq 1 } \) and \( k \in \mathbb{ N } _ 0 \) \[ \sum_{n = 0}^{k} x^{n} = \frac{1 - x^{k + 1}}{1 - x} \]
Let \( S = \sum_{n = 0}^{k} x^{n} \), then \( x S - S = x^{k + 1} - x^{0} \) therefore \( S = \frac{x^{k + 1} - 1}{x - 1} \), so also \( S = \frac{1 - x^{k + 1}}{1 - x} \) by factoring -1 from the numerator and the denominator, as needed
Or instead it can be proven inductively as follows, in the base case we get \[ \sum _ { n = 0 } ^ 0 x ^ n = x ^ 0 = 1 = \frac{1 - x ^ 1}{1 - x} \] assume it holds true for \( j \in \mathbb{ N } _ 0 \) and we want to prove that it holds true for \( j + 1 \) \[ \begin{align} \sum _ { n = 0 } ^ { j + 1 } x ^ n &= \sum _ { n = 0 } ^ { j } x ^ n + x ^ { j + 1 } \\ &= \frac{1 - x ^ { j + 1 } }{ 1 - x} + \frac{x ^ { j + 1 } \left( 1 - x \right) }{ 1 - x} \\ &= \frac{1 - x ^ { j + 1 } + x ^ { j + 1 } \left( 1 - x \right)}{ 1 - x} \\ &= \frac{1 - x ^ { j + 2 } }{1 - x} \end{align} \] as needed.

Note that when \( x = 1 \) the sum simply yields \( k + 1 \).

Series
Given a sequence \( \left( a _ n \right) _ { n = 1 } ^ \infty \) and defining \( S _ k = \sum _ { i = 1 } ^ k a _ k \) then we define \[ \sum _ { i = 0 } ^ \infty a _ i := \lim _ { n \to \infty } S _ n \]

Note that because it is defined as a limit we can ask if \( \sum _ { i = 0 } ^ \infty a _ n \) converges or not.

Harmonic Series Diverges
\[ \sum _ { n = 1 } ^ \infty \frac{1}{n} \] diverges to \( \infty \)
We claim that \( S _ { 2 ^ k } \ge \frac{k + 2 }{2} = \frac{k}{2} + 1 \) for all \( k \in \mathbb{ N } _ 0 \), for the base case we see that \( S _ { 2 ^ 0 } = S _ 1 = \frac{1}{1} \ge \frac{0}{2} + 1 \) so it holds true, now suppose its true for some \( j \in \mathbb{ N } _ 0 \) we want to prove that \( S _ { 2 ^ { j + 1 } } \ge \frac{j + 1 }{2} + 1 \) \[ \begin{align} S _ { 2 ^ { k + 1 } } &= S _ { 2 ^ k } + \sum _ { m = 2 ^ k + 1 } ^ { 2 ^ { k + 1 } } \frac{1}{m} \\ &\ge \left( \frac{k }{2} + 1 \right) + \sum _ { m = 2 ^ k + 1 } ^ { 2 ^ { k + 1 }} \frac{1}{m} \\ &\ge \left( \frac{k }{2} + 1 \right) + \sum _ { m = 2 ^ k + 1 } ^ { 2 ^ { k + 1 } }\frac{1}{2 ^ { k + 1 } } \\ &\ge \left( \frac{k }{2} + 1 \right) + \sum _ { m = 2 ^ k + 1 } ^ { 2 ^ { k + 1 } } \frac{1}{2 ^ { k + 1 } } \\ &= \left( \frac{k }{2} + 1 \right) + 2 ^ k \left( \frac{1}{2 ^ { k + 1 } } \right) \\ &= \left( \frac{k }{2} + 1 \right) + \frac{1}{2} \\ &= \frac{k + 1 }{2} + 1 \end{align} \] Thus since \( S _ { 2 ^ k } \ge \frac{k}{2} + 1 \) and \( \lim _ { k \to \infty } \frac{k}{2} + 1 = \infty \) then we know that \( \sum _ { k \to \infty } S _ { 2 ^ k } = \infty \) and thus since a subsequence \( S _ n \) diverges then we know that \( S _ n \) diverges
Geometric Power Series
for \( \left | x \right | < 1 \) we have \[ \sum_{n = 0}^{\infty} x^{n} = \frac{1}{1 - x} \]

By definition we know that \( \sum_{n = 0}^{\infty} x^{n} = \lim_{k \to \infty} \sum_{n = 0}^{k} x^{n} \)\( = \) \( \lim_{k \to \infty} \frac{1 - x^{k + 1}}{1 - x} \)

exponent limit lemma if \( \left | x \right | < 1 \) then \( \lim_{k \to \infty} x^{k + 1} \) becomes zero and we get \( \lim_{k \to \infty} \frac{1 - x^{k + 1}}{1 - x} = \frac{1}{1 - x} \)
Series of Halves is Bounded
\[ \sum _ { n = 0 } ^ \infty \frac{1}{2} ^ n = \frac{1}{1 - \frac{1}{2} } = 2 \]
In the above proposition take \( x = \frac{1}{2} \).
geometric sum with incrementing factor
Given \( t \in \mathbb{R}^{\ne 0} \), the following holds: \[ \sum_{i = 1}^{n} i t^{i} = t \frac{1 - t^{n}}{\left ( 1 - t \right )^{2}} - n \frac{t^{n + 1}}{1 - t} \]
We want to find the sum defined by
\( S \left ( t \right ) := t + 2 t^{2} + 3 t^{3} + \ldots + \left ( n - 1 \right ) t^{n - 1} + n t^{n} \),
But then we know that
\( t \cdot S \left ( t \right ) = t^{2} + 2 t^{3} + 3 t^{4} + \ldots + \left ( n - 1 \right ) t^{n} + n t^{n + 1} \)
\( \iff \)
\( S \left ( t \right ) - t S \left ( t \right ) = t + t^{2} + t^{3} + \ldots + t^{n} - n t^{n + 1} \)
\( \iff \)
\( \left ( 1 - t \right ) S \left ( t \right ) = t + t^{2} + t^{3} + \ldots + t^{n} - n t^{n + 1} \)
Now since we know that \( t + t^{2} + t^{3} + \ldots + t^{n} = t \frac{1 - t^{n}}{1 - t} \), then we have
\( \left ( 1 - t \right ) S \left ( t \right ) = t \frac{1 - t^{n}}{1 - t} - n t^{n + 1} \)
\( \iff \)
\( S \left ( t \right ) = \frac{t \frac{1 - t^{n}}{1 - t} - n t^{n + 1}}{1 - t} \)
\( \iff \)
\( S \left ( t \right ) = t \frac{1 - t^{n}}{\left ( 1 - t \right )^{2}} - n \frac{t^{n + 1}}{1 - t} \)
geometric series with incrementing factor
Suppose that \( \left | t \right | \lt 1 \), then \[ \sum_{i = 1}^{\infty} i t^{i} = \frac{t}{\left ( 1 - t \right )^{2}} \]

We know that \( \sum_{i = 1}^{\infty} i t^{i} = \lim_{n \to \infty} \sum_{i = 1}^{n} i t^{i} \) \( = \) \( \lim_{n \to \infty} t \frac{1 - t^{n}}{\left ( 1 - t \right )^{2}} - n \frac{t^{n + 1}}{1 - t} \)

Now since \( \left | t \right | \lt 1 \), then \( \lim_{n \to \infty} t^{n} = 0 \) and \( \lim_{n \to \infty} n t^{n + 1} = 0 \), therefore \( \lim_{n \to \infty} t \frac{1 - t^{n}}{\left ( 1 - t \right )^{2}} - n \frac{t^{n + 1}}{1 - t} = \frac{t}{\left ( 1 - t \right )^{2}} \) as needed.

Convergent Series Implies Sequence goes to Zero
If \( \sum _ { n = 1 } ^ \infty a _ n \) is convergent then \( \lim _ { n \to \infty } a _ n = 0 \)
Let \( \left( S _ n \right) _ { n = 1 } ^ \infty \) be so defined such that \( S _ n = \sum _ { i = 0 } ^ n a _ i \), observe that \( S _ n - S _ { n - 1 } = a _ n \) for all \( n \ge 2 \) so then \[ \lim _ { n \to \infty } a _ n = \lim _ { n \to \infty } \left( s _ n - s _ { n - 1 } \right) = \lim _ { n \to \infty } s _ n - \lim _ { n \to \infty } s _ { n - 1 } = 0 \]
Cauchy Criterion for Series
Suppose that \( \sum _ { n = 1 } ^ \infty a _ n \) is a series, then the following are equivalent:
  1. The series converges
  2. \( \forall \epsilon \in \mathbb{ R } ^ + , \exists N \in \mathbb{ N } _ 0 \text{ st } \forall n \in \mathbb{ N } _ 0 , n \ge N \implies \lvert \sum _ { k = n + 1 } ^ \infty a _ k \rvert \lt \epsilon \)
  3. \( \forall \epsilon \in \mathbb{ R } ^ + , \exists N \in \mathbb{ N } _ 0 \text{ st } \forall n, m \in \mathbb{ N } _ 0 , n, m \ge N \implies \lvert \sum _ { k = n + 1 } ^ m a _ k \rvert \lt \epsilon \)

Note that 2 informally says that tails converge and are arbitrarily small, and 3 essentially says that \( S _ n \) is a cauchy sequence because \( \sum _ { k = n + 1 } ^ m a _ k = S _ m - S _ n \)
