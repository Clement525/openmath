Harmonic Number
We define the \( n \) -th harmonic number as \[ H _ n := 1 + \frac{1 }{2} + \frac{1}{3} + \ldots + \frac{1}{n} = \sum _ { k = 1 } ^ n \frac{1}{k} \]
Closed form of a Geometric Summation
For any \( x \in \mathbb{ R } ^ { \neq 1 } \) and \( k \in \mathbb{ N } _ 0 \) \[ \sum_{n = 0}^{k} x^{n} = \frac{1 - x^{k + 1}}{1 - x} \]
Let \( S = \sum_{n = 0}^{k} x^{n} \), then \( x S - S = x^{k + 1} - x^{0} \) therefore \( S = \frac{x^{k + 1} - 1}{x - 1} \), so also \( S = \frac{1 - x^{k + 1}}{1 - x} \) by factoring -1 from the numerator and the denominator, as needed
Or instead it can be proven inductively as follows, in the base case we get \[ \sum _ { n = 0 } ^ 0 x ^ n = x ^ 0 = 1 = \frac{1 - x ^ 1}{1 - x} \] assume it holds true for \( j \in \mathbb{ N } _ 0 \) and we want to prove that it holds true for \( j + 1 \) \[ \begin{align} \sum _ { n = 0 } ^ { j + 1 } x ^ n &= \sum _ { n = 0 } ^ { j } x ^ n + x ^ { j + 1 } \\ &= \frac{1 - x ^ { j + 1 } }{ 1 - x} + \frac{x ^ { j + 1 } \left( 1 - x \right) }{ 1 - x} \\ &= \frac{1 - x ^ { j + 1 } + x ^ { j + 1 } \left( 1 - x \right)}{ 1 - x} \\ &= \frac{1 - x ^ { j + 2 } }{1 - x} \end{align} \] as needed.

Note that when \( x = 1 \) the sum simply yields \( k + 1 \).

Cauchy Criterion for Series
Suppose that \( \sum _ { n = 1 } ^ \infty a _ n \) is a series, then the following are equivalent:
  1. The series converges
  2. \( \forall \epsilon \in \mathbb{ R } ^ + , \exists N \in \mathbb{ N } _ 0 \text{ st } \forall n \in \mathbb{ N } _ 0 , n \ge N \implies \lvert \sum _ { k = n + 1 } ^ \infty a _ k \rvert \lt \epsilon \)
  3. \( \forall \epsilon \in \mathbb{ R } ^ + , \exists N \in \mathbb{ N } _ 0 \text{ st } \forall n, m \in \mathbb{ N } _ 0 , n, m \ge N \implies \lvert \sum _ { k = n + 1 } ^ m a _ k \rvert \lt \epsilon \)

Note that 2 informally says that tails converge and are arbitrarily small, and 3 essentially says that \( S _ n \) is a cauchy sequence because \( \sum _ { k = n + 1 } ^ m a _ k = S _ m - S _ n \)

Cauchy Condensation Test
Suppose that \( \left( a _ n \right) : \mathbb{ N } _ 1 \to \mathbb{ R } ^ + \) is monotone decreasing then \[ \sum _ { n = 1 } ^ \infty a _ n \lt \infty \iff \sum _ { n = 0 } ^ \infty 2 ^ n a _ { 2 ^ n } \lt \infty \]
Let's define \( S _ n = \sum _ { k = 1 } ^ n a _ k \) and \( T _ n = \sum _ { k = 0 } ^ n 2 ^ k a _ { 2 ^ k } \).

Assume that \( \sum _ { n = 0 } ^ \infty 2 ^ n a _ { 2 ^ n } \) converges, which by definition means that \( \left( T _ n \right) \) converges and is therefore bounded above, which means there is some \( M \) such that for any \( i \in \mathbb{ N } _ 0, T _ i \le M \), we'll use this fact to show that \( \left( S _ n \right) \) is bounded above by showing that it has a subsequence which is bounded above. Namely we will prove that \[ S _ { 2 ^ n - 1 } \le t _ { n - 1 } \le M \]

Base case \( n = 1 \), \( S _ { 2 ^ 1 - 1 } = S _ 1 = a _ 1 = 2 ^ 0 a _ { 2 ^ 0 } = T _ 0 \le M \) as needed. Now asssume that it holds true for \( n \in \mathbb{ N } _ 1 \), which is to say that \( S _ { 2 ^ n - 1 } \le T _ { n - 1 } \le M \) and now let's prove that \( S _ { 2 ^ { n + 1 } - 1 } \le T _ { n } \le M \)

Note that in the induction step the following fact will be useful, which says for any \( n \in \mathbb{ N } _ 1 \) we have that \[ T _ { n } - T _ { n - 1 } = 2 ^ n a _ { 2 ^ n } \] Now continuing on, we note the following

\[ \begin{align} S _ { 2 ^ { n + 1 } - 1 } &= S _ { 2 ^ n - 1 } + \sum _ { i = 2 ^ n } ^ { 2 ^ { n + 1 } - 1 } a _ i\\ &\le T _ { n - 1 } + \sum _ { i = 2 ^ n } ^ { 2 ^ { n + 1 } - 1 } a _ i\\ &\le T _ { n - 1 } + 2 ^ n a _ { 2 ^ n } \\ &= T _ { n - 1 } + \left( T _ n - T _ { n - 1 } \right) \\ &= T _ { n } \le M \end{align} \] Note that going from the second line to the third comes from the fact that \( \left( a _ n \right) \) is decreasing which means for any \( x \in \left\{ 2 _ n, \ldots, 2 ^ { n + 1 } - 1 \right\} \) that \( a _ { 2 ^ n } \ge a _ x \) additionally the the size of that set is \( 2 ^ { n + 1 } - 1 - 2 ^ n + 1 = 2 ^ n \left( 2 - 1 \right) = 2 ^ n \), which allow us to upper bound that summation by \( 2 ^ n a _ { 2 ^ n } \), which shows the induction step true. Thus we've proven that \( S _ n \) is bounded above, and so by the MCT it converges which is to say that \( \sum _ { n = 1 } ^ \infty a _ n \lt \infty \)

Now let's assume that \( \sum _ { n = 0 } ^ \infty 2 ^ n a _ { 2 ^ n } \) converges, and show that the other one does as well, we'll follow a similar structure that in we show that \( T _ n \) is bounded above. Specifically we will show that \[ a _ 1 + 2 \sum _ { i = 2 } ^ { 2 ^ n } a _ n \ge T _ n \] Since \( \left( S _ n \right) \) is bounded above by some \( M \in \mathbb{ R } \) then by adding \( a _ 1 \) to both sides we obtain \( 2 S _ { 2 ^ n } \ge a _ 1 + T _ n \) which is to say that \( 2 M \ge a _ 1 + T _ n \) which shows that \( T _ n \) is bounded above, and since it is increasing we would know that it converges by the MCT.

For the base case of \( n = 0 \) we have \( a _ 1 + 2 a _ 2 \ge a _ 1 = T _ 0 \) so it holds true. Now let \( n \in \mathbb{ N } _ 0 \) and suppose that \( a _ 1 + 2 \sum _ { i = 2 } ^ { 2 ^ n } a _ n \ge T _ n \) we need to prove that \( a _ 1 + 2 \sum _ { i = 2 } ^ { 2 ^{ n + 1 } } a _ n \ge T _ { n + 1 } \), then notice that \[ \begin{align} a _ 1 + 2 \sum _ { i = 2 } ^ { 2 ^ { n + 1 } } a _ n &= a _ 1 + 2 \sum _ { i = 2 } ^ { 2 ^ n } a _ n + 2 \sum _ { i = 2 ^ n + 1 } ^ { 2 ^ { n + 1 } } a _ n \\ &\ge T _ n + 2 \sum _ { i = 2 ^ n + 1 } ^ { 2 ^ { n + 1 } } a _ n \\ &\ge T _ n + 2 \left( 2 ^ n a _ { 2 ^ { n + 1 } } \right) \\ &= T _ n + 2 ^ { n + 1 } a _ { 2 ^ { n + 1 } } \\ &= T _ n + \left( T _ { n + 1 } - T _ n \right) \\ &= T _ { n + 1 } \end{align} \] In this case the 3rd line follows by a similar argument to the previous induction step, but this team using the fact that for all \( x \in \left\{ 2 ^ n + 1 , \ldots , 2 ^ { n + 1 } \right\} \) we have \( a _ x \ge a _ { 2 ^ { n + 1 } } \) because \( \left( a _ n \right) \) is monotone decreasing.

Alternating Sequence
A sequence is alternating if it has the form \( \left( -1 \right) ^ n a _ n \) or \( \left( -1 \right) ^ \left( n + 1 \right) a _ n \) where \( a _ n \ge 0 \) for all \( n \in \mathbb{ N } _ 1 \).

This characterization simply states that consecutive terms change sign.

Alternating Series
A series \( \sum _ { n = 1 } ^ \infty a _ n \) is said to be alternating if \( \left( a _ n \right) \) is alternating.
Leibniz Alternating Series Test
Suppose that \( \left( a _ n \right) : \mathbb{ N } _ 1 \to \mathbb{ R } \) is a monotone decreasing sequence and \( \lim _ { n \to \infty } a _ n = 0 \) then the alternating series \[ \sum _ { n = 1 } ^ \infty \left( -1 \right) ^ n a _ n \] converges
Absolutely Convergent Series
A series \( \sum _ { n = 1 } ^ \infty a _ n \) is said to be absolutely convergent if the series \( \sum _ { n = 1 } ^ \infty \left\lvert a _ n \right\rvert \) converges.
Conditionally Convergent Series
A series \( \sum _ { n = 1 } ^ \infty a _ n \) is said to be conditionally convergent if it converges but \( \sum _ { n = 1 } ^ \infty a _ n \) does not.

Recall that \( \sum _ { n = 1 } ^ \infty \frac{\left( -1 \right) ^ { n + 1 } }{n} \) converges to \( \ln \left( 2 \right) \) and is conditionally convergent because the harmonic series diverges.

Series Rearrangement
A rearrangment of a series \( \sum _ { n = 1 } ^ \infty \) is another series \[ \sum _ { n = 1 } ^ \infty a _ { \pi \left( n \right) } \] where \( \pi \) is a permutation of \( \mathbb{ N } _ 1 \)

The above definition formalizes the idea of taking a series and then looking at it with the same terms in a different order. Also note that the definition of a series is the limit of the partial sums, which is an entirely different thing than the \( + \) operation, thus we have no idea whether or not the new series will have the same limit, or even exist.
