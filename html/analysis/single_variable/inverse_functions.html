increasing function
given `A sube RR`, a function `f: A to RR` is increasing if given `x, y in A` with `x < y` we have `f(x) le f(y)`. An increasing function is also known as a non-decreasing function.
strictly increasing function
given `A sube RR`, a function `f: A to RR` is strictly increasing if given `x, y in A` with `x < y` we have `f(x) lt f(y)`.
decreasing function
given `A sube RR`, a function `f: A to RR` is decreasing if given `x, y in A` with `x < y` we have `f(x) ge f(y)`. A decreasing function is also known as a non-increasing function.
strictly decreasing function
given `A sube RR`, a function `f: A to RR` is decreasing if given `x, y in A` with `x < y` we have `f(x) gt f(y)`
monotone function
a function defined on a subset of `RR` is said to be monotone if and only if it is non-increasing or non-decreasing
strictly monotone function
a function defined on a subset of `RR` is said to be monotone if and only if it is stictly increasing or strictly decreasing
invertible function
Let `f: X to Y` be a function, if there is a function `g: Y to X` such that `g(f(x)) = x` for all `x in X` and `f(g(y)) = y` for all `y in Y`
a function is invertible iff it is bijective
