Negative Real Number
We say that \( x \in \mathbb{ R } \) is negative when \[ x \lt 0 \]
Positive Real Number
We say that \( x \in \mathbb{ R } \) is positive when \[ 0 \lt x \]
A Negative Number is Smaller than a Positive One
Suppose that \( a \lt 0 \) and that \( b \gt 0 \) then \[ a \lt b \]
This follows because \( \lt \) is transitive.
Multiplying by -1 Inverts the Sign
Suppose \( x \in \mathbb{ R } \) is positive iff \( -x \) is negative
Observe that \( x \ge 0 \) iff \( x + \left( -x \right) \ge 0 + \left( -x \right) \) iff \( 0 \ge -x \) as needed.
Negative Zero is Zero
For \( 0 \in \mathbb{ R } \) we have that \( -0 = 0 \)
Since \( 0 \) is an identity element with respect to \( + \) then \( x + 0 = x \), specifically for \( x = 0 \) by definition this means that \( -0 = 0 \) as needed.
A Positive Number is Greater than the Negative Version of Itself
Suppose \( x \in \mathbb{ R } \) where \( x \ge 0 \) then \( x \ge -x \)
If \( x = 0 \) then \( 0 = -0 \) so trivially \( 0 \ge -0 \). On the other hand if \( x \gt 0 \) then \( 0 \ge -x \) therefore \( x \ge -x \)
Multiplying by -1 Changes the Inequality Direction
For any \( x, y \in \mathbb{ R } \) \[ x \ge y \iff -x \le -y \]
Observe that \( x \ge y \) is the same as \( x - y \ge 0 \) is the same as \( 0 \ge y - x \) is the same as \( -x \le -y \) as needed.
Absolute Value
the absolute value function \( \left \lvert \cdot \right \rvert : \mathbb{R} \to \mathbb{R}^{\ge 0} \) is defined by
\( \left | x \right | = \left \lbrace \begin{matrix} x \text{ if } x \ge 0 \\ - x \text{ if } x \lt; 0 \end{matrix} \right . \)
Absolute Value Differs By Sign
\[ \forall x \in \mathbb{R} , x = \left | x \right | \vee x = - \left | x \right | \]
Let \( x \in \mathbb{R} \) if \( x \gt; 0 \), then \( x = \left | x \right | \) by definition, if \( x \lt; 0 \), then \( \left | x \right | = - x \), or rather \( x = - \left | x \right | \)
Absolute Value Equals Max of Itself and it's Negation
Let \( x \in \mathbb{ R } \) then \[ \lvert x \rvert = \max \left( x, -x \right) \]

Recall the definition of \( \max \left( x, -x \right) \) which is that it's value is \( x \) if \( x \ge -x \) and \( -x \) if \( x \le -x \). Suppose that \( x \ge 0 \) then \( \lvert x \rvert = x \) and since \( x \ge -x \) then we know that \( \max \left( x, -x \right) = x \) and so in this case we see that \( \lvert x \rvert = \max \left( x \right) \).

Now in the case that \( x \lt 0 \) then \( -x \gt 0 \) and therefore \( -x \gt x \) therefore \( \max \left( x, -x \right) = -x \) and since \( x \lt 0 \) then \( \lvert x \rvert = -x \).

Therefore it's been verified in any case so that \( \lvert x \rvert = \max \left( x, -x \right) \).

Multiplication in Absolute Value
\[ \left \lvert x \cdot y \mid = \right \rvert x \left \lvert \cdot \mid y \right \rvert \]
  • If both \( x , y \) are positive then \( \left | x \cdot y \right | = x \cdot y = \left | x \right | \cdot \left | y \right | \)
  • if both \( x , y \) are negative, then their product is positive so it follows as above
  • if one of \( x , y \) is positive and the other negative, without loss of generality assume that \( x \lt; 0 \) and \( y \gt; 0 \)then their product is negative so \( \left | x \cdot y \right | = - 1 \cdot x \cdot y = \left | x \right | \cdot \left | y \right | \)
  • if any one of \( x , y \) is zero, then \( \left | x \cdot y \right | = 0 = \left | x \right | \cdot \left | y \right | \)
Value Between Positive and Negative Equivalence
Let \( a, x \in \mathbb{ R } \) where \( x \ge 0 \) \[ -x \le a \le x \iff \lvert a \rvert \le x \]

\( \implies \) we want to prove that \( \lvert a \rvert \le x \). Case one, if \( a \ge 0 \) then \( \lvert a \rvert = a \) and since we already know that \( a \le x \) then we obtain that \( \lvert a \rvert \le x \) as needed. In the second case if \( a \lt 0 \) then \( \lvert a \rvert = -a \) so we want to show that \( -a \le x \) which is the same as \( a \ge -x \) which we know by assumption as well.

\( \impliedby \) Suppose that \( \lvert a \rvert \le x \) if \( a \ge 0 \) then \( \lvert a \rvert = a \) and so we're really assuming that \( a \le x \) therefore \( -a \ge -x \) since \( a \ge -a \) then by chaining inequalities we get \( x \ge a \ge -a \ge -x \) which implies \( x \ge a \ge -x \) as needed. Now for the case of \( a \le 0 \) then we've assumed that \( -a \le x \) now \( -a \) is positive and by symmetrically following our previous case with \( -a \) in place of \( a \) we see that \( x \ge -a \ge a \ge -x \) and so \( x \ge a \ge -x \) again, as needed.

Generalized Triangle Inequality
Suppose that \( X \subseteq \mathbb{ R } \) is finite then \[ \left\lvert \sum X \right\rvert \le \sum _ { x \in X } \left\lvert x \right\rvert \]
We prove it true by induction based on the size of \( X \). Base case \( \left\lvert X \right\rvert = 2 \) is covered by the regular triangle inequality. Now suppose that it holds true for \( \left\lvert Y \right\rvert = k \) we'll show it holds true for \( \left\lvert Z \right\rvert = k + 1 \), let \( z \in Z \), then we have \( \left\lvert \sum \left( Z \setminus \left\{ z \right\} \right) \right\rvert \le \sum _ { a \in Z \setminus \left\{ z \right\} } \left\lvert a \right\rvert \) since \( \left\lvert Z \setminus \left\{ z \right\} \right\rvert = k \). Now by the regular triangle inequality we obtain that \[ \begin{align} \left\lvert \sum Z \right\rvert &= \left\lvert \sum \left( Z \setminus \left\{ z \right\} \right) + z \right\rvert \\ &\le \left\lvert \sum \left( Z \setminus \left\{ z \right\} \right) \right\rvert + \left\lvert z \right\rvert \\ &\le \sum _ { a \in Z \setminus \left\{ z \right\} } \left\lvert a \right\rvert + \left\lvert z \right\rvert \\ &= \sum _ { z \in Z } \left\lvert z \right\rvert \end{align} \]
