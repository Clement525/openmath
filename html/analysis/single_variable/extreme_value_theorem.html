upper bound
Given `S sube P` where `P` is a partial order, then `u in P` is said to be an upper bound iff `AA s in S, s le u`. we say that `S` is bounded above by `u`
lower bound
Given `S sube P` where `P` is a partial order , then `l in P` is said to be an lower bound iff `AA s in S, l le s`, in this case we say that `S` is bounded below by `l`.
bounded
A set `S sube P` where `P` is a partial order, then `P` is said to be bounded if it is bounded above and bounded below

infimum
The infimum of a subset `S` of a partially ordered set `P` is the greatest element in `P` that is a lower bound of `S` and is denoted by `text(inf)(S)`. Symbolically this means that `AA s in S, s ge text(inf)(S)`, and if `k` is a lower bound, then `k le text(inf)(S)`
supremum
The supremum of a subset `S` of a partially ordered set `P` is the least element in `P` that is a upper bound of `S` and is denoted by `text(sup)(S)`. Symbolically this means that `AA s in S, s le text(sup)(S)`, and if `k` is a upper bound, then `k ge text(sup)(S)`
greater than inf implies not a lower bound
Suppose that `text(inf)(S) lt x`, then `x` is not an lower bound of `S`

Suppose that `text(inf)(S) lt x`, then for the sake of contradiction assume that `x` is a lower bound of `S`, then `text(inf)(S) ge x` which is a contradiction.

less than sup implies not an upper bound
if `x lt text(sup)(S)`, then `x` is not a upper bound of `S`

Suppose that `text(sup)(S) gt x`, then for the sake of contradiction assume that `x` is a upper bound of `S`, then `text(sup)(S) le x` which is a contradiction.

least upper bound property
We say that a partially ordered set `X` has the least upper bound property if every non-empty subset of `X` that has an upper bound, has a least upper bound in `X`. Symbolically that is, if `S sube X` and `S ne O/`, if `S` has an upper bound, then `text(sup)(S) in S`
