Closed form of a Geometric Series
\( \sum_{n = 0}^{k} x^{n} = \frac{1 - x^{k + 1}}{1 - x} \)
Let \( S = \sum_{n = 0}^{k} x^{n} \), then \( x S - S = x^{k + 1} - x^{0} \) therefore \( S = \frac{x^{k + 1} - 1}{x - 1} \), so also \( S = \frac{1 - x^{k + 1}}{1 - x} \) by factoring -1 from the numerator and the denominator, as needed
Geometric Power Series
for \( \left | x \right | < 1 \) we have \( \sum_{n = 0}^{\infty} x^{n} = \frac{1}{1 - x} \)

By definition we know that \( \sum_{n = 0}^{\infty} x^{n} = \lim_{k \to \infty} \sum_{n = 0}^{k} x^{n} \)\( = \) \( \lim_{k \to \infty} \frac{1 - x^{k + 1}}{1 - x} \)

exponent limit lemma if \( \left | x \right | < 1 \) then \( \lim_{k \to \infty} x^{k + 1} \) becomes zero and we get \( \lim_{k \to \infty} \frac{1 - x^{k + 1}}{1 - x} = \frac{1}{1 - x} \)
geometric sum with incrementing factor
Given \( t \in \mathbb{R}^{\ne 0} \), show that
\( \sum_{i = 1}^{n} i t^{i} = t \frac{1 - t^{n}}{\left ( 1 - t \right )^{2}} - n \frac{t^{n + 1}}{1 - t} \)
We want to find the sum defined by
\( S \left ( t \right ) := t + 2 t^{2} + 3 t^{3} + \ldots + \left ( n - 1 \right ) t^{n - 1} + n t^{n} \),
But then we know that
\( t \cdot S \left ( t \right ) = t^{2} + 2 t^{3} + 3 t^{4} + \ldots + \left ( n - 1 \right ) t^{n} + n t^{n + 1} \)
\( \iff \)
\( S \left ( t \right ) - t S \left ( t \right ) = t + t^{2} + t^{3} + \ldots + t^{n} - n t^{n + 1} \)
\( \iff \)
\( \left ( 1 - t \right ) S \left ( t \right ) = t + t^{2} + t^{3} + \ldots + t^{n} - n t^{n + 1} \)
Now since we know that \( t + t^{2} + t^{3} + \ldots + t^{n} = t \frac{1 - t^{n}}{1 - t} \), then we have
\( \left ( 1 - t \right ) S \left ( t \right ) = t \frac{1 - t^{n}}{1 - t} - n t^{n + 1} \)
\( \iff \)
\( S \left ( t \right ) = \frac{t \frac{1 - t^{n}}{1 - t} - n t^{n + 1}}{1 - t} \)
\( \iff \)
\( S \left ( t \right ) = t \frac{1 - t^{n}}{\left ( 1 - t \right )^{2}} - n \frac{t^{n + 1}}{1 - t} \)
geometric series with incrementing factor
Suppose that \( \left | t \right | \lt 1 \), then
\( \sum_{i = 1}^{\infty} i t^{i} = \frac{t}{\left ( 1 - t \right )^{2}} \)

We know that \( \sum_{i = 1}^{\infty} i t^{i} = \lim_{n \to \infty} \sum_{i = 1}^{n} i t^{i} \) \( = \) \( \lim_{n \to \infty} t \frac{1 - t^{n}}{\left ( 1 - t \right )^{2}} - n \frac{t^{n + 1}}{1 - t} \)

Now since \( \left | t \right | \lt 1 \), then \( \lim_{n \to \infty} t^{n} = 0 \) and \( \lim_{n \to \infty} n t^{n + 1} = 0 \), therefore \( \lim_{n \to \infty} t \frac{1 - t^{n}}{\left ( 1 - t \right )^{2}} - n \frac{t^{n + 1}}{1 - t} = \frac{t}{\left ( 1 - t \right )^{2}} \) as needed.
