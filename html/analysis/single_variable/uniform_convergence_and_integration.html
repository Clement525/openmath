Integral Convergence
Let \( \left( f _ n \right) : \mathbb{ N } _ 1 \to C \left[ a, b \right] \) such that \( f _ n \stackrel{unif}{\to } f \) and fix \( c \in \left[ a, b \right] \) then the new sequence of functions \( F _ n : \left[ a, b \right] \to \mathbb{ R } \) \[ F _ n \left( x \right) := \int _ c ^ x f _ n \left( t \right) dt \] converges uniformly on \( \left[ a, b \right] \) to the function \[ F \left( x \right) := \int _ c ^ x f \left( t \right) dt \]
