Integral Convergence
Let \( \left( f _ n \right) : \mathbb{ N } _ 1 \to C \left[ a, b \right] \) such that \( f _ n \stackrel{unif}{\to } f \) and fix \( c \in \left[ a, b \right] \) then the new sequence of functions \( F _ n : \left[ a, b \right] \to \mathbb{ R } \) \[ F _ n \left( x \right) := \int _ c ^ x f _ n \left( t \right) dt \] converges uniformly on \( \left[ a, b \right] \) to the function \[ F \left( x \right) := \int _ c ^ x f \left( t \right) dt \]
Derivative Convergence
Suppose that \( f _ n \) is a sequence of continuously differentiable functions on \( \left[ a, b \right] \) such that \( f _ n ^ \prime \) converges uniformly to a function \( g \) and there is a point \( c \in \left[ a, b \right] \) such that \( \lim _ { n \to \infty } f _ n \left( c \right) = \gamma \) exists. Then \( f _ n \) converges uniformly to a differentiable function \( f \) with \( f \left( c \right) = \gamma \) and \( f ^ \prime = g \)
