Sequence Convergence
Let \( \left( x_n \right) : \mathbb{R} \to \mathbb{R} \) be a sequence and let \( x \in \mathbb{ R } \), then we say that \( \left( x _ n \right) \) converges to \( x \) and write \( \left( x _ n \right) \to x \) when the following holds true \[ \forall \epsilon \in \mathbb{ R }^+, \exists n \in \mathbb{ N } _ 0 \text{ such that } \forall n \in \mathbb{ N } _ 0, n \ge N \implies | x _ n - x | \lt \epsilon \]

Note that you may come across various other notations for this definition in other resources, the main ones to note will be: \( x _ n \to x \) as \( n \to \infty \), or even more shortened as \( x _ n \to x \), sometimes limit notation will be used: \( \lim _ { n \to \infty } x _ n = x \) or just \( \lim x _ n = x \)

Convergent Sequence Doesn't Require Strict Inequality
\( \left( a _ n \right) \to L \) is equivalent to \[ \forall \epsilon \in \mathbb{ R } ^ +, \exists N \in \mathbb{ N } _ 0, \text{ st } \forall n \in \mathbb{ N } _ 0, n \ge N \implies \left\lvert a _ n - L \right\rvert \le \epsilon \]
\( \implies \) Suppose \( \left( a _ n \right) \to L \), now let \( \epsilon \in \mathbb{ R } \), and consider \( \frac{\epsilon}{2} \), then since \( \left( a _ n \right) \to L \) we get some \( N \in \mathbb{ N } _ 0 \) such that for every \( n \ge N \) we have \( \left\lvert a _ n - L \right\rvert \lt \frac{\epsilon}{2} \le \epsilon \) as needed. \( \impliedby \) Assume the other direction, and we want to prove \( \left( a _ n \right) \to L \) so let \( \epsilon \in \mathbb{ R } ^ + \) therefore we get an \( N \in \mathbb{ N } _ 0 \) such that for all \( n \ge N \) we have \( \left\lvert a _ n - L \right\rvert \le \frac{\epsilon }{2} \lt \epsilon \) as needed.
Limit Notation
Suppose that \( x _ n \to x \) then we write \[ \lim _ { n \to \infty } x _ n := x \]

Note that if you have an arbitrary sequence \( \left( y _ n \right) \) which you don't know if it converges to anything and you write \( \lim _ { n \to \infty } y _ n = L \) for some \( L \in \mathbb{ R } \), this is two assertions, first that \( \left( y _ n \right) \) converges to something, and the value it converges to is \( L \).

Condensing what we've proven above, we've shown that \[ \forall k \in \mathbb{ N } _ 1, \exists N _ k \in \mathbb{ N } _ 1, \sigma _ { m _ k } : \mathbb{ N } _ 1 \to \mathbb{ N } _ 1 \text{ st } \forall n \ge N _ 2, P \left( k, \sigma _ m \left( n \right) \right) \]

Where \( P \left( k, v \right) := \left\lvert x _ v - L \right\rvert \lt \frac{1}{k} \), and since \( \sigma _ { m _ k } \) defines a subsequence then it is strictly increasing, therefore there exists an ordered sequence \( \left( \alpha _ 1 \le \alpha _ 2 \le \alpha _ 3 \le \ldots \right) \) such that \( P \left( k , \alpha _ k \right) \) which is to say that \( \left\lvert x _ { \alpha _ k } - L \right\rvert \lt \frac{1}{k} \) so that the following subsequence works \[ \left( x _ { \alpha _ 1 }, x _ { \alpha _ 2 }, \ldots \right) \] or in another notation with \( \sigma \left( n \right) = \alpha _ n \) we are using the subsequence \( \left( x _ { \sigma \left( n \right) } \right) \)

which is obtained by using the fact that we can insert \( 0 = - a _ i + a _ i \) into the absolute value and then use the triangle inequality to break it, the general result is proven using induction using this fact in the induction step.

Let \( \left( a _ n \right) \) be a sequence such that \( \lim _ { N \to \infty } \sum _ { n = 1 } ^ N \lvert a _ n - a _ { n + 1 } \rvert \lt \infty \), show that \( \left( a _ n \right) \) is cauchy.

Cauchy Sequence Implies Subsequence with Convergent Differences
If \( \left( x _ n \right) _ { n = 1 } ^ \infty \) is cauchy, show that it has a subsequence \( \left( x _ { \sigma \left( n \right) } \right) \) such that \[ \sum _ { k = 1 } ^ \infty \lvert x _ { \sigma \left( k \right) } - x _ { \sigma \left( k + 1 \right) } \rvert \lt \infty \]

Let's construct this subsequence, for any \( k \in \mathbb{ N } _ 1 \) consider \( \epsilon = \frac{1}{2 ^ k} \), then we get an \( N _ k \) such that for all \( n, m \ge N _ k, \lvert a _ n - a _ m \rvert \le \frac{1}{2 ^ k} \), and consider the sequence \( x _ { N _ 1 }, x _ { N _ 2 }, x _ { N _ 3 }, \ldots \), we'll show it satisfies \( \alpha \).

Let \( j \in \mathbb{ N } _ 1 \), and consider \( N _ j, N _ { j + 1 } \), in the case that \( N _ j \ge N _ { j + 1 } \) then note that \( \lvert x _ { N _ j } - x _ { N _ { j + 1 } } \rvert \le \frac{1}{2 ^ { j + 1 } } \le \frac{1}{2 ^ j} \) (where \( n = N _ j, m = N _ { j + 1 } \)), in the case that \( N _ j \lt N _ { j + 1 } \) we also get that \( \lvert x _ { N _ j } - x _ { N _ { j + 1 } } \rvert \le \frac{1}{2 ^ j} \) so \( \alpha \) is satisfied for this subsequence. Therefore \[ \sum _ { k = 1 } ^ \infty \lvert x _ { N _ k } - x _ { N _ { k + 1 } } \rvert \le \sum _ { k = 1 } ^ \infty \frac{1}{2 ^ k} = 1 \] as needed.

Monotone Increasing Sequence
A sequence \( \left( a _ n \right) _ { n = 1 } ^ \infty \) is said to be monotone increasing whenever we have: \[ a _ n \le a _ { n + 1 } \] for any \( n \in \mathbb{ N } _ 1 \)
Monotone Decreasing Sequence
A sequence \( \left( a _ n \right) _ { n = 1 } ^ \infty \) is said to be monotone decreasing whenever we have: \[ a _ n \ge a _ { n + 1 } \] for any \( n \in \mathbb{ N } _ 1 \)
Bounded Above and Monotone Increasing Converges to Supremum
A monotone increasing sequence that is bounded above converges to \( L = \sup \left( \operatorname{ im } \left( \left( a _ n \right) \right) \right) \)

Since \( \left( a _ n \right) _ { n = 1 } ^ \infty \) is bounded above, then so is \( \operatorname{ im } \left( \left( a _ n \right) \right) \subseteq \mathbb{ R } \) therefore since \( \mathbb{ R } \) has the least upper bound property, we know that \( \sup \left( \operatorname{ im } \left( \left( a _ n \right) \right) \right) \) exists, call it \( L \) and we'll prove that \( \left( a _ n \right) \) converges there.

Let \( \epsilon \in \mathbb{ R } ^ + \) by definition of the supremum we know that \( L - \epsilon \) is not an upper bound for \( A \), in other words there exists some element \( a _ N \in \operatorname{ im } \left( \left( a _ n \right) \right) \) such that \( L - \epsilon \lt a _ N \)

Since \( \left( a _ n \right) \) is increasing, we know that for any \( n \in \mathbb{ N } , n \ge N , a _ N \le a _ n \), moreover \( L \) is an upperbound so that we get \[ L - \epsilon \lt a _ N \le a _ n \le L \lt L + \epsilon \implies L - \epsilon \lt a _ n \lt L + \epsilon \] therefore \( \lvert a _ n - L \rvert \lt \epsilon \), therefore \( \lim _ { n \to \infty } a _ n = L \) as needed.

Bounded Below and Monotone Decreasing Converges to Infimum
A monotone decreasing sequence that is bounded below converges to \( L = \inf \left( \operatorname{ im } \left( \left( a _ n \right) \right) \right) \)
If \( \left( a _ n \right) \) is decreasing and bounded below by \( B \), then the sequence \( \left( - a _ n \right) \) is increasing and bounded above by \( - B \). Thus the sequence \( \left( -a _ n \right) _ { n = 1 } ^ \infty \) has limit \( L = \sup \left( \operatorname{ im } \left( \left( - a _ n \right) \right) \right) \) recall that \[ \sup \left( \operatorname{ im } \left( \left( - a _ n \right) \right) \right) = - \inf \left( \left\{ -x : x \in \operatorname{ im } \left( \left( - a _ n \right) \right) \right\} \right) = - \inf \left( \operatorname{ im } \left( a _ n \right) \right) \] So that \( -L = \inf \left( \operatorname{ im } \left( \left( a _ n \right) \right) \right) \). Then note that \( \lim _ { n \to \infty } - a _ n = L \) implies that \( -L = \lim _ { n \to \infty } a _ n \) so we conclude that \[ \lim _ { n \to \infty } a _ n = \inf \left( \operatorname{ im } \left( \left( a _ n \right) \right) \right) \] as needed.

Inequality From Sequence to Limit

Nested Intervals
Suppose that \( I _ n = \left[ a _ n, b _ n \right] \) are nonempty closed intervals such that \( I _ { n + 1 } \subseteq I _ n \) for every \( n \in \mathbb{ N } _ 1 \), then \[ \bigcap _ { n \in \mathbb{ N } _ 1 } I _ n \neq \emptyset \]

Since \( \left[ a _ { n + 1 }, b _ { n + 1 } \right] \subseteq \left[ a _ n, b _ n \right] \) then we know that \( a _ n \le a _ { n + 1 } \le b _ { n + 1 } \le b _ n \) for any \( n \in \mathbb{ N } _ 1 \) and therefore we conclude that \( \left( a _ n \right) \) is monotone increasing and that \( \left( b _ n \right) \) is monotone decreasing. Also note that \( a _ n \le b _ n \le b _ 1 \) and \( a _ 1 \le a _ n \le b _ n \) so that by transitivity we obtain \( a _ n \le b _ 1 \) and \( a _ 1 \le b _ n \) which is to say that \( a _ n \) is bounded above and that \( b _ n \) is bounded below.

Therefore from both versions of the montone convergence theorem we see that \( a _ n \) and \( b _ n \) converge to \( a := \sup \left( \operatorname{ im } \left( \left( a _ n \right) \right) \right) , b := \inf \left( \operatorname{ im } \left( b _ n \right) \right) \) respectively and from the above proposition we see that \( a \le b \)

By the definition of the supremum and infimum, we also have that \[ a _ k \le a \le b \le b _ k \] for all \( k \in \mathbb{ N } _ 1 \) meaning that \( a, b \) belongs to \( I _ k \), so that \( \left\{ a, b \right\} \subseteq \bigcap _ { n \in \mathbb{ N } _ 1 } I _ n \) and so it must not be empty.
