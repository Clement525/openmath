sample space
A sample space `Omega` is a non-empty set
event
Given a sample space `Omega`, we say that any subset `E sube Omega` is an event
probability measure
Given a sample space `Omega` and a function `P: Omega to [0, 1]` , then we say that `P` is a probability measure if the following holds
  • `P(Omega)` `= 1`
  • `AA E_1, E_2, ... sube Omega` such that `E_1, E_2, ...` are all pairwise disjoint
    `P(bigcup_(i = 1)^oo E_i) = sum_(i = 1)^oo P(E_i) `
finite pairwise disjoint probability
Suppose that `A_1, A_2, ..., A_n` are pairwise disjoint, then `P(A_1 uu ... uu A_n) = sum_(i = 1)^n P(A_i)`
In the definition of a probability measure, for `i in {1, ..., n}` we set `E_i = A_i` and then for any `j > n` we set `E_j = O/`, then we can see that `E_1, E_2, ...` is pairwise disjoint, and thus
`P(A_1 uu A_2 uu ... uu A_n)` `=` `P(A_1 uu A_2 uu ... uu A_n uu O/ uu O/ uu ...)`
`=` `P(bigcup_(i = 1)^oo E_i)`
`=` `sum_(i = 1)^oo P(E_i)`
`=` `sum_(i = 1)^n P(E_i) + P(O/) + P(O/) + P(O/) + ... `
`=` `sum_(i = 1)^n P(E_i) + 0 + 0 + 0 + ... `
`=` `sum_(i = 1)^n P(A_i)`
probability of a single point with equally likely outcomes is zero
Suppose that our sample space is `[0, 1]` and there is some `c in [0, 1]` such that for any `x in [0, 1]`, `P({x}) = c`, then `c = 0`
Suppose that `c ne 0`, and note that `P(Omega) = P( [{:0, 1 :}] ) = sum_(x in [0, 1]) P(x) = sum_(x in [0, 1]) c`, but note that no matter how small the value of `c`, since we are summing it uncountably many times with itself the sum always goes to infinity, but at the same time, `P(Omega) = 1` and thus we have a contradiction, so `c = 0`.
random variable
Suppose that `Omega` is a sample space. A random variable is a function `X: Omega to RR`
inverse image notation
Suppose that `X: Omega to RR ` is a random variable and that `y in RR`, then we define the string of characters `X = y` to be the set `X^(-1)({y})`
conditional probability
Suppose that `A, B sube Omega`, with `P(B) gt 0`, then we define the conditional probability of `A` given `B` as
`P(A | B) = (P(A nn B)) / (P(B))`
