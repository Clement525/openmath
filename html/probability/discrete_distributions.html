Discrete Random Variable
A random variable \( X \) is discrete if \[ \sum _ { x \in \mathbb{ R } } P \left( X = x \right) = 1 \]
Probability Function
For a discrete random variable \( X \), it's probability function is the function \( p _ X : \mathbb{ R } \to [ 0,1 ] \) defined by \[ p _ X \left( x \right) := P \left( X = x \right) \]
