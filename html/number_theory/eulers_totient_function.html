Set of Coprime Integers
Suppose that \( n \in \mathbb{ N } _ 1 \) then we define \[ \operatorname{ cop } \left( n \right) := \left\{ k \in \left[ 1, \ldots , n \right] : \gcd \left( k, n \right) = 1 \right\} \]
Euler's Totient Function
We define it as \( \phi : \mathbb{ N } _ 1 \to \mathbb{ C } \) as \[ \phi \left( n \right) := \left\lvert \operatorname{ cop } \left( n \right) \right\rvert \]
GCD Set
We define \( G \left( d, n \right) : \mathbb{ N _ 1 } \times \mathbb{ N } _ 1 \to \mathbb{ N } _ 1 \) as \[ G \left( d, n \right) := \left\{ k \in \left[ 1, \ldots , n \right]: \operatorname{ gcd } \left( k, n \right) = d \right\} \]
Bijection between Cops and G's
Let \( d, n \in \mathbb{ N } _ 1 \) such that \( d \mid n \), \[ \left\lvert C _ d \right\rvert = \left\lvert G \left( \frac{n}{d} , n \right) \right\rvert \]
The GCD Sets form a Partition
The collection \( \left\{ G \left( i, n \right): i \in \left[ 1, \ldots n \right] \right\} \) is a partition of \( \left[ 1, \ldots , n \right] \)
