Order Exists iff There is a Power that Yields 1
As per title.
Suppose that \( \operatorname{ o } _ n \left( a \right) \) exists, then we have at least one power that yields one. On the other hand if we have some power that yields one, then the set \( \left\{ k \in \mathbb{ N } _ 1 : \left( a ^ k ~\%~ n \right) = 1 \right\} \) is non-empty so that the min exists.
Order Exists iff Coprime
We have that \( a \in \operatorname{ cop } \left( n \right) \) iff \( o \left( a, n \right) \) exists

If \( a \in \operatorname{ cop } \left( n \right) \) then by euler's theorem we know that \( a ^ { \phi \left( n \right) } \equiv 1 \pmod{ n } \) therefore the order exists.

On the other hand if the oreder exists, then we have some \( a ^ k \equiv 1 \pmod{ n } \), now recall that \( ax \equiv 1 \pmod{ n } \) has a solution if and only if \( a \in \operatorname{ cop } \left( n \right) \), but note we have a solution, becuase if \( k = 1 \) then \( x = 1 \) works, otherwise if \( k \ge 2 \) then \( x = a ^ { k - 1 } \) works therefore we have a solution in any case so that \( a \in \operatorname{ cop } \left( n \right) \) as needed.

Primitive Roots are a Subset of Coprimes
\[ \operatorname{ pr } \left( n \right) \subseteq \operatorname{ cop } \left( n \right) \]
If \( a \in \operatorname{ pr } \left( n \right) \) then \( a \) has an order, but \( a \) has an order iff \( a \in \operatorname{ cop } \left( n \right) \), note that not everything that has an order is a primitive root.
Every Prime number has a Primitive Root
For any \( p \in \mathbb{ P } \), \( \operatorname{ pr } \left( p \right) \neq \emptyset \)

Suppose for the sake of contradiction that it had no primitive roots therefore for any \( a \in \left[ 1, \ldots , p - 1 \right] \) that \( d := o \left( a, p \right) \lt \phi \left( p \right) = p - 1 \), but at the same time we know that \( d \mid p - 1 \).

Now focusing on \( a \) for a moment, we see that for any \( k \in \left[ 1, \ldots d \right] \) we have \[ o \left( a ^ k, n \right) = \frac{d}{\gcd \left( d, k \right) = d} \iff \gcd \left( d, k \right) = 1 \] so there are at least \( phi \left( d \right) \) elements (using powers of \( a \) ) that have order \( d \), and noting that since \( a, a ^ 2, \ldots , a ^ d \) are distinct mod p, then so are these elements.

By lagranges theorem, the polynomial \( x ^ d - 1 \) has at most \( d \) roots modulo \( p \)

Connection Between Primitive Roots of \( p ^ k \) and 2 \( p ^ k \)
Let \( p \in \mathbb{ P } _ 3 \), \( k \in \mathbb{ N } _ 1 \) and \( a \in \operatorname{ pr } \left( p ^ k \right) \) \[ a \in \operatorname{ pr } \left( 2 p ^ k \right) \iff a \in \mathbb{ Z } _ \operatorname{ odd } \]

Suppose that \( a \in \operatorname{ pr } \left( 2 p ^ k \right) \) then \( a \in \operatorname{ cop } \left( 2 p ^ k \right) \), therefore \( a \) cannot be even, so that it must be odd.

Now suppose that \( a \in \mathbb{ Z } _ \operatorname{ odd } \) but for the sake of contradiction that \( a \notin \operatorname{ pr } \left( 2 p ^ k \right) \) therefore \( m := \operatorname{ o } \left( a, 2p ^ k \right) \lt \phi \left( 2 p ^ k \right) = \phi \left( p ^ k \right) \) and \( a ^ m \equiv 1 \pmod{ 2 p ^ k } \) then we have \( a ^ m \equiv 1 \pmod{ p ^ k } \), but \( a \in \operatorname{ pr } \left( p ^ k \right) \) therefore \( \operatorname{ o } \left( a, p ^ k \right) = \phi \left( p ^ k \right) \) but we have that \( m \lt \phi \left( p ^ k \right) \) which is impossible because the order is the smallest power the yields one, but we've found a smaller one.
