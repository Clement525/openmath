Primitive Roots are a Subset of Coprimes
\[ \operatorname{ pr } \left( n \right) \subseteq \operatorname{ cop } \left( n \right) \]
If \( a \in \operatorname{ pr } \left( n \right) \) then \( a \) has an order, but \( a \) has an order iff \( a \in \operatorname{ cop } \left( n \right) \), note that not everything that has an order is a primitive root.
Every Prime number has a Primitive Root
For any \( p \in \mathbb{ P } \), \( \operatorname{ pr } \left( p \right) \neq \emptyset \)

Suppose for the sake of contradiction that it had no primitive roots therefore for any \( a \in \left[ 1, \ldots , p - 1 \right] \) that \( d := o \left( a, p \right) \lt \phi \left( p \right) = p - 1 \), but at the same time we know that \( d \mid p - 1 \).

Now focusing on \( a \) for a moment, we see that for any \( k \in \left[ 1, \ldots d \right] \) we have \[ o \left( a ^ k, n \right) = \frac{d}{\gcd \left( d, k \right) = d} \iff \gcd \left( d, k \right) = 1 \] so there are at least \( phi \left( d \right) \) elements (using powers of \( a \) ) that have order \( d \), and noting that since \( a, a ^ 2, \ldots , a ^ d \) are distinct mod p, then so are these elements.

By lagranges theorem, the polynomial \( x ^ d - 1 \) has at most \( d \) roots modulo \( p \)

A Number to the Phi n over Two is Congruent to 1 When it is not a Primitive Root
Let \( n \in \mathbb{ N } _ 3 \) such that \( \operatorname{ pr } \left( n \right) = \emptyset \), and \( c \in \operatorname{ cop } \left( n \right) \), then we have \[ c ^ \frac{\phi \left( n \right) }{2} \equiv 1 \pmod{ n } \]

Since \( \operatorname{ pr } \left( n \right) = \emptyset \) then \( n \notin \left\{ 2, 4, p ^ k, 2 p ^ k \right\} \) for some \( p \in \mathbb{ P } _ 3 \) and \( k \in \mathbb{ N } _ 1 \).

We will deal with the case when \( n = 2 ^ k \) for \( k \in \mathbb{ N } _ 3 \), we know that \( c ^ \phi \left( n \right) \equiv 1 \pmod{ n } \) and that \( \phi \left( n \right) = \phi \left( 2 ^ k \right) = 2 ^ { k - 1 } \) and therefore \( o _ n \left( c \right) = 2 ^ j \) for some \( j \in \left[ 1, \ldots , k - 1 \right] \), clearly we see \( c \notin \emptyset = \operatorname{ pr } \left( n \right) \) and so we know that \( j \neq k - 1 \) because in that case we would have that \( o _ n \left( c \right) = \phi \left( 2 ^ { k - 1 } \right) \) which implies that \( \operatorname{ pr } \left( n \right) \neq \emptyset \) which is a contradiction. This proves that \( o _ n \left( c \right) = 2 ^ j \mid 2 ^ { k + 1 } = \frac{\phi \left( n \right) }{2} \) and therefore we have that \( c ^ \frac{\phi \left( n \right) }{2} \equiv 1 \pmod{ n } \).
