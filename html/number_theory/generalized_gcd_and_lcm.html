Generalized GCD
Suppose that \( S \subseteq \mathbb{ Z } \) is finite that is not all zero, then \[ \gcd \left( S \right) \] is the largest positive integer that divides everything in \( S \).
GCD of GCD's Is the Same as Their Union
Suppose that \( T \subseteq \mathcal{ P } \left( \mathbb{ N } _ 1 \right) \) such that \( T \) is finite and for every \( S \in T, S \) is finite, then we have \[ \gcd \left( \left\{ \gcd \left( S \right): S \in T \right\} \right) = \gcd \left( \bigcup T \right) \]
Generalized Bezout Identity
There exists integers \( x _ 1, \ldots , x _ n \) such that \[ a _ 1 x _ 1 + a _ 2 x _ 2 + \cdots + a _ n x _ n = \gcd \left( a _ 1, a _ 2, \ldots , a _ n \right) \]

We prove this by induction, so for the base case we use the basic bezout identity to see that \( gcd \left( a _ 1, a _ 2 \right) = a _ 1 x _ 1 + a _ 2 x _ 2 \) .

Now suppose it holds true for \( k \in \mathbb{ N } _ 2 \) and we'll show that it holds true for \( k + 1 \), we have \[ \begin{align} \gcd \left( a _ 1, \ldots a _ { k + 1 } \right) &= \gcd \left( \gcd \left( a _ 1, \ldots , a _ k \right), a _ { k + 1 } \right) \\ &= \gcd \left( \sum _ { i = 1 } ^ { k } a _ i x _ i, a _ { k + 1 } \right) \\ &= b \left( \sum _ { i = 1 } ^ {k } a _ i x _ i \right) + c a _ { k + 1 } \\ &= \sum _ { i = 1 } ^ { k + 1 } d _ i a _ i \end{align} \] The first line uses this fact, to get to the the second line we used the induction hypothesis and to get to the third line we used the basic bezout for 2 elements.

Divisible by Relatively Prime Numbers Means Divisible by Their Product
Suppose that \( \gcd \left( a _ 1, \ldots a _ n \right) = 1 \) and \( k \in \mathbb{ Z } \) then \[ \left( \forall i \in \left[ 1, \ldots n \right] , a _ i \mid k \right) \implies \prod _ { i = 1 } ^ { n } a _ i \mid k \]
Since \( \gcd \left( a _ 1, \ldots, a _ n \right) = 1 \) then there exists some \( x _ 1, \ldots , x _ n \) such that \( \sum _ { i = 1 } ^ { n } a _ i x _ i = 1 \), so that \( \sum _ { i = 1 } ^ { n } a _ i x _ i k = k \) we know that each \( a _ i \mid k \) so that there is some \( j _ i \in \mathbb{ Z } \) such that \( k = a _ i j _ i \) so we have \[ \sum _ { i = 1 } ^ { n } a _ i x _ i k = \sum _ { i = 1 } ^ { n } \]
