Quadratic Residue
Let \( n \in \mathbb{ N } _ 1 \) and \( c \in \mathbb{ Z } \) we say that \( c \) is a quadratic residue when the congruence \[ x ^ 2 \equiv c \pmod{ n } \] has a solution, we define the collection of quadratic residues as \( \operatorname{ QR } \left( n \right) \)

Being a quadratic residue means "you have a square root"

Eulers Criterion
Let \( p \in \mathbb{ P } _ 3 \) and \( c \in \mathbb{ Z } \) such that \( p \nmid c \) then \( c \) is a quadratic residue if and only if \[ c ^ \frac{p - 1}{2} \equiv 1 \pmod{ p } \]
Legendre Symbol
Let \( p \in \mathbb{ P } _ 3 \) and \( c \in \mathbb{ Z } \), then we define the legendre symbol as: \[ \operatorname{ leg } \left( c, p \right) := \begin{cases} 0 & \text{ if } p \mid c \\ 1 & \text{ if } p \nmid c \text{ and } c \in \operatorname{ QR } \left( n \right) \\ -1 & \text{ if } p \nmid c \text{ and } c \notin \operatorname{ QR } \left( n \right) \\ \end{cases} \]
Quadratic Residue iff Legendre Symbol is One
Let \( c \in \mathbb{ Z } \) and \( p \in \mathbb{ P } _ 3 \) then \[ \operatorname{ leg } \left( c, p \right) = 1 \] if and only if there is some \( x _ 0 \in \mathbb{ Z } \) such that \[ x _ 0 ^ 2 \equiv c \pmod{ p } \]
Quadratic Residues mod p Come in Pairs
Let \( p \in \mathbb{ P } ^ 3 \), \( c \in \operatorname{ cop } \left( p \right) \) and suppose that \( x _ 0 \in \mathbb{ Z } \) is a solution to \( x ^ 2 \equiv c \pmod{ p } \) then there are exactly 2 solutions to this equation given by \( \pm x _ 0 \)

Clearly if \( x _ 0 \) is a solution then so is \( - x _ 0 \) because \( \left( -x _ 0 \right) ^ 2 = x ^ 2 \equiv c \pmod{ p } \). These solutions are indeed unique as they would be the same iff \( x _ 0 \equiv 0 \pmod{ p } \) which is impossible as that would imply that \( c \equiv 0 \pmod{ p } \) and \( c \in \operatorname{ cop } \left( p \right) \), therefore it must be that \( x _ 0 \neq 0 \), now \( x _ 0 \in \left[ 1, \ldots , p - 1 \right] \) and \( p - x _ 0 \left[ 1, \ldots , p - 1 \right] \) since \( p - 1 \) is even, then it's impossible that \( x _ 0 = p - x _ 0 \) (which implies that \( p \) is even), therefore we must have that \( x _ 0 \neq p - x _ 0 \) and therefore \( x _ 0 \not \equiv p - x _ 0 \equiv - x _ 0 \pmod{ p } \) so these two solutions are distinct solutions.

Suppose there were another solution given by \( s \) so that \( s ^ 2 \equiv c \equiv x _ 0 ^ 2 \), then we have \[ x _ 0 ^ 2 - s ^ 2 \equiv 0 \iff \left( x _ 0 + s \right) \left( x _ 0 - s \right) \equiv 0 \pmod{ p } \] and so we deduce that \( p \mid x _ 0 + s \) or \( p \mid x _ 0 - s \) which is to say that either \( s \equiv -x _ 0 \pmod{ p } \) or \( s \equiv x _ 0 \pmod{ p } \) which means that any other solution is congruent to one of \( x _ 0, - x _ 0 \), therefore we conclude that there are exactly two solutions.

Supposing we had a solution to \( x ^ 2 \equiv c \pmod{ n } \) , then these would also be solutions to \( x ^ 2 \equiv c \pmod{ p _ i ^ { \alpha _ i } } \) for any \( i \in \left[ 1, \ldots , l \right] \)

So now suppose that we have a solution for each of the individual \( x ^ 2 \equiv c \pmod{ p _ i ^ { \alpha _ i } } \) call each solution instance \( s _ i \), now setup a new system for each \( i \in \left[ 1, \ldots , l \right] \) as \[ y \equiv s _ i \pmod{ p _ i ^ { \alpha _ i } } \] by applying the crt we obtain a solution \( \bar x \) to this system which is unique mod \( n \), moreover \( { \bar x } ^ 2 \equiv s _ i ^ 2 \pmod{ p _ i ^ { \alpha _ i } } \).

When a Quadratic Congruence has a Solution mod p Squared
Let \( p \in \mathbb{ P } _ 3 \) and \( c \in \operatorname{ cop } \left( p \right) \), then \[ x ^ 2 \equiv c \pmod{ p ^ 2 } \] has solutions if and only if \( \operatorname{ leg } \left( c, p \right) = 1 \)

\( \implies \) Suppose that we have a solution \( x _ 0\) so that \( x _ 0 ^ 2 \equiv c \pmod{ p ^ 2 } \) then \( x _ 0 ^ 2 \equiv c \pmod{ p } \) thus \( \operatorname{ leg } \left( c, p \right) = 1 \)

\( \impliedby \) Now suppose that \( \operatorname{ leg } \left( c, p \right) = 1 \) therefore we have a solution \( x _ 0 \in \mathbb{ Z } \) to \( x ^ 2 \equiv c \pmod{ p } \) therefore for some \( m \in \mathbb{ Z } \) we have that \[ x _ 0 ^ 2 = mp + c \] if \( p \mid m \) then \( x _ 0 ^ 2 \equiv c \pmod{ p } \) and we would be done, so in the other case when \( p \nmid m \) then we will have to find a new solution to the equation, consider \( y _ 0 = x _ 0 + np \), let's see if we can construct a solution using this form. If we attempt this solution we see that \[ \left( x _ 0 + np \right) ^ 2 = x _ 0 ^ 2 + 2np + n ^ 2 p ^ 2 \equiv x _ 0 ^ 2 + 2np \equiv c + mp + 2x _ 0 np \pmod{ p ^ 2 } \] Note that \( m + 2x _ 0 np = p \left( m + 2x _ 0 n \right) \), so now we require an \( n \) such that \[ 2 x _ 0 + m \equiv 0 \pmod{ p } \] but recall that \( \operatorname{ leg } \left( c, p \right) = 1 \) thus \( c \in \operatorname{ cop } \left( p \right) \) thus \( x _ 0 \in \operatorname{ cop } \left( n \right) \) for if it were not the case then we would have \( c \notin \operatorname{ cop } \left( p \right) \) which would be a contradiction as \( x _ 0 ^ 2 \equiv c \pmod{ p } \), this shows that \( x _ 0 \) has an inverse, similarly \( 2 \) has an inverse as \( 2 \in \operatorname{ cop } \left( p \right) \) since \( p \in \mathbb{ P } _ 3 \) so therefore we have \[ 2x _ 0 n + m \equiv 0 \pmod{ p } \iff n \equiv -m \left( 2 x _ 0 \right) ^ { -1 } \] and thus we can always find a value of \( n \) that will work, meaning \( y _ 0 = x _ 0 + np \) is a solution to \( x _ 2 \equiv c \pmod{ p } \) as needed.
