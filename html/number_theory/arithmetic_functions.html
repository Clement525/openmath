Arithmetic Function
An arithmetic function \( f \) is a function such that \( \operatorname{ dom } \left( f \right) = \mathbb{ N } _ 1 \)
Divisor Counting Function
Let \( n \in \mathbb{ N } _ 1 \) and define \[ \tau \left( n \right) := \left\lvert \left\{ d \in \mathbb{ N } _ 1: d \mid n \right\} \right\rvert \]
Equation for the Divisor Counting Function
Suppose that \( n \in \mathbb{ N } _ 2 \) so that \( n = \prod _ { i \in \mathbb{ N } _ 1 } p _ i ^ { \alpha _ i } \) then \[ \tau \left( n \right) = \prod _ { i \in \mathbb{ N } } \left( \alpha _ i + 1 \right) \]
Divisor Sum Function
Let \( n \in \mathbb{ N } _ 1 \) and define \[ \sigma \left( n \right) = \sum \left\{ d \in \mathbb{ N } _ 1: d \mid n \right\} \] the above notation is a sum fold
Equation for the Divisor Sum Function
Suppose that \( n \in \mathbb{ N } _ 2 \) so that \( n = \prod _ { i = 1 } ^ k p _ i ^ { \alpha _ i } \) then \[ \sigma \left( n \right) = \prod _ { i = 1 } ^ { k } \frac{p _ i ^ { \alpha _ i } - 1 }{p _ i - 1} \]
Multiplicative
Suppose that \( f: \mathbb{ N } _ 1 \to C \subseteq \mathbb{ C } \) is arithmetic, then we say that \( f \) is multiplicative if for every coprime \( n, m \in \mathbb{ N } _ 1 \) we have \[ f \left( n m \right) = f \left( n \right) f \left( m \right) \]
Totally Multiplicative
Suppose that \( f: \mathbb{ N } _ 1 \to C \subseteq \mathbb{ C } \) is arithmetic, then we say that \( f \) is totally multiplicative if for \( n, m \in \mathbb{ N } _ 1 \) we have \[ f \left( n m \right) = f \left( n \right) f \left( m \right) \]

When the Constant Function is Multiplicative
The constant function is multiplicative if and only if \( c \in \left\{ 0, 1 \right\} \)
The Product of Two Multiplicative Functions is Multiplicative
Suppose that \( f, g: \mathbb{ N } _ 1 \to C \) be multiplicative then so is \( f \cdot g \)
Nowhere Zero
We say that a function is nowhere zero if \( 0 \notin \operatorname{ im } \left( f \right) \)
The Quotient of Two Multiplicative Functions is Multiplicative
Suppose that \( f, g: \mathbb{ N } _ 1 \to C \) be multiplicative and that \( g \) is nowhere zero then \( \frac{f}{g} \) is multiplicative
Two Multiplicative Functions are Equal if they Agree on Prime Powers
Suppose that \( f, g: \mathbb{ N } _ 1 \to C \) are multiplicative such that \( f \left( 1 \right) = g \left( 1 \right) \) then if for any \( p \in \mathbb{ P } \) and \( k \in \mathbb{ N } _ 1 \) we have \[ f \left( p ^ k \right) = g \left( p ^ k \right) \] then \( f = g \)
Sum of Reciprocals of Divisors Equation
Show that \[ \frac{\sigma \left( n \right) }{n} = \sum _ { d \mid n } \frac{1}{d} \]
Since both functions are multiplicative, then we confirm that the align at \( 1 \) which they do. Then let \( p \in \mathbb{ P } \) and \( k \in \mathbb{ N } _ 1 \) then we have \[ \begin{align} \sum _ { d \mid p ^ k } \frac{1}{d} &= \sum _ { j = 0 } ^ { k } \frac{1}{p ^ j} \\ &= \frac{1}{p ^ k} \left( \sum _ { j = 0 } ^ { k } p ^ k \right) \\ &= \frac{\sigma \left( p ^ k \right) }{p ^ k} \end{align} \] therefore the two functions are equal
The Sum of a Multiplicative Function over Divisors of a Number is Multiplicative

Dirichlet Convolution
Suppose \( f, g: \mathbb{ N } \to C \) are arithmetic then the Dirichlet Convolution of \( f \) and \( g \) denoted \( f * g \) is defined as \[ \left( f * g \right) \left( n \right) := \sum _ { d \mid n } f \left( d \right) g \left( \frac{n}{d} \right) \]
