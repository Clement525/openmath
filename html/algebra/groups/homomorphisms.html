Group Homomorphism
Suppose that \( \left( G, \star \right) \) and \( \left( H, \cdot \right) \) are groups, then we say that a group homomorphism from \( \left( G, \star \right) \) to \( \left( H, \cdot \right) \) is a function \( \phi: G \to H \) such that for any \( u, v \in G \) \[ \phi \left( u \star v \right) = \phi \left( u \right) \cdot \phi \left( v \right) \]

Note that a homomorphism is an isomorphism without the bijective requirement.

The order of The Mapped Element Divides the Original
Suppose that \( \lvert g \rvert \) is finite, then \( \lvert \phi \left( g \right) \rvert \) divides \( \lvert g \rvert \)
Suppose \( g \) has order \( n \), so that \( g ^ n = e \), now that is to say that \( e = \phi \left( e \right) = \phi \left( g ^ n \right) = \phi \left( g \right) ^ n \), then \( \lvert \phi \left( g \right) \rvert \) divides \( n = \lvert g \rvert \) as needed.
Kernel of a Homomorphism
The kernel of a homomorphism from the group \( G \) to the identity \( e _ H \) of \( H \) is the set \[ \left\{ g \in G: \phi \left( g \right) = e_H \right\} = \phi ^ { -1 } \left( \left\{ e _ H \right\} \right) \]
Right Iterated Binary Operation
Suppose that \( \left( \star, G, \star \right) \) is a group, then we define \[ \operatorname{ right\_iter } \left( \star, g, n \right) = \begin{cases} g & \text{ if } n = 0 \\ \star \left( g, \operatorname{ right\_iter } \left( \star, g, n - 1 \right) \right) &\text{ if } n \in \mathbb{ N } _ 1 \end{cases} \]
Left Iterated Binary Operation
Suppose that \( \left( G, \star \right) \) is a group, then we define \[ \operatorname{ left\_iter } \left( \star, g, n \right) = \begin{cases} g & \text{ if } n = 0 \\ \star \left( \operatorname{ left\_iter } \left( \star, g, n - 1 \right), g \right) &\text{ if } n \in \mathbb{ N } _ 1 \end{cases} \]

The above can be generalized to operate on sequences of elements of \( g \).

The Image of a Homomorphism is Abelian iff It's Kernel Contains the Subgroup Generated by \( x y x ^ { -1 } y ^ { -1 } \)
Suppose that \( \phi : G \to H \) is a group homomorphism. Then \( \operatorname{ im } \left( \phi \right) \) is abelian iff \( [ G, G ] \subseteq \operatorname{ ker } \left( \phi \right) \)

\( \implies \) Suppose that \( \operatorname{ im } \left( \phi \right) \) is abelian, we want to prove that \( [ G, G ] \subseteq \operatorname{ ker } \left( \phi \right) \), we already know that \( [ G, G ] \) is the smallest subgroup containing the set \( \left\{ x y x ^ { -1 } y ^ { -1 } : x, y \in G \right\} \) since we already know that \( \operatorname{ ker } \left( \phi \right) \) is a subgroup of \( G \), then we just have to show it contains the set, and it would automatically become a superset as needed.

Let \( x y x ^ { -1 } y ^ { -1 } \) be in the set, to show that this element is also in \( \operatorname{ ker } \left( \phi \right) \) we have to show that \( \phi \left( x y x ^ { -1 } y ^ { -1 } \right) = e _ H \), we note that \( \operatorname{ im } \left( \phi \right) \) was abelian and a homomorphism, thus \[ \begin{align} \phi \left( x y x ^ { -1 } y ^ { -1 } \right) &= \phi \left( x \right) \phi \left( y \right) \phi \left( x ^ { -1 } \right) \phi \left( y ^ { -1 } \right) \\ &= \phi \left( x \right) \phi \left( y \right) \phi \left( x \right) ^ { -1 } \phi \left( y \right) ^ { -1 } \\ &= \phi \left( x \right) \phi \left( x \right) ^ { -1 } \phi \left( y \right) \phi \left( y \right) ^ { -1 } \\ &= e _ H e _ H \\ &= e _ H \end{align} \] Note that pulling the power out of a homomorphism is justified on line 2

\( \impliedby \) Now we're assuming that \( [ G, G ] \subseteq \operatorname{ ker } \left( \phi \right) \) and we want to show that \( \operatorname{ im } \left( \phi \right) \) is abelian so let \( a, b \in \operatorname{ im } \left( \phi \right) \), so that there is some \( x, y \in G \) such that \( a = \phi \left( x \right), b = \phi \left( y \right) \). Let's prove \( a b = b a \):

Firstly by definition \( [ G, G ] \) contains the set \( \left\{ xyx ^ { -1 } y ^ { -1 } : x, y \in G \right\} \) therefore since \( \operatorname{ ker } \left( \phi \right) \) is a superset of \( [ G, G ] \) it also contains that set, in other words \( x y x ^ { -1 } y ^ { -1 } \in \operatorname{ ker } \left( \phi \right) \), that is \( \phi \left( x y x ^ { -1 } y ^ { -1 } \right) = e _ H \), well \( \phi \) is a homomorphism: \( \phi \left( x \right) \phi \left( y \right) \phi \left( x \right) ^ { -1 } \phi \left( y \right) ^ { -1 } = e _ H \), the by multiplying each side on the right we have \( \phi \left( x \right) \phi \left( y \right) = \phi \left( y \right) \phi \left( x \right) \), well that is \( a b = b a \) as needed.
