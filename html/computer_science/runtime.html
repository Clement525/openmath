Absolutely Dominated
Let \( f, g : \mathbb{ N } _ 0 \to \mathbb{ R } ^ { \ge 0 } \), we say that \( g \) is absolutely dominated by \( f \) iff for all \( n \in \mathbb{ N } _ 0, g \left( n \right) \le f \left( n \right) \)
Squared Dominates Linear
Let \( f \left( n \right) = n ^ 2 \) and \( g \left( n \right) = n\) show that \( g \) is absolutely dominated by \( f \)
Absolutely Dominated up to a Constant Factor
Let \( f, g : \mathbb{ N } _ 0 \to \mathbb{ R } ^ { \ge 0 } \), we say that \( g \) is absolutely dominated up to a constant factor by \( f \) iff there exists \( c \in \mathbb{ R } ^ + \) such that \( g \) is absolutely dominated by \( c f \)
Eventually Dominated
Let \( f, g : \mathbb{ N } _ 0 \to \mathbb{ R } ^ { \ge 0 } \), we say that \( g \) is eventually dominated by \( f \) iff there exists some \( n _ 0 \in \mathbb{ R } ^ + \) such that \( \forall n \in \mathbb{ N } ^ +, n \ge n _ 0 \) then we know \( g \left( n \right) \le f \left( n \right) \)
Lower Degree Polynomial plus Constant is Eventually Dominated by Higher Degree
Let \( f \left( n \right) = n ^ 2 \) and \( g \left( n \right) = n + 90 \)
Eventually Dominated up to a Constant Factor
Let \( f, g : \mathbb{ N } _ 0 \to \mathbb{ R } ^ { \ge 0 } \), we say that \( g \) is eventually dominated by \( f \) iff there exists some \( c \in \mathbb{ R } ^ + \) such that \( g \) is eventually dominated by \( c f \)
Big-O
Suppose we have \( f: \mathbb{N}_{0} \to \mathbb{R}^{\ge 0} \), then we define the set \( \mathcal{ O } \left( f \right) \) as all functions that are eventually dominated up to a constant factor by \( f \)
Big-O of
Let \( f, g : \mathbb{ N } _ 0 \to \mathbb{ R } ^ { \ge 0 } \), we say that \( g \) is big-o of \( f \) iff \( g \in \mathcal{ O } \left( f \right) \)

Note that if \( g \) is big-o of \( f \), then we know that \( f \) is an asymptotic upper bound for \( g \)

Omega
Suppose we have \( f: \mathbb{N}_{0} \to \mathbb{R}^{\ge 0} \), then we define the set \( \Omega \left( f \right) \) as the set of all functions \( g : \mathbb{N}_{0} \to \mathbb{R}^{\ge 0} \) such that \[ \exists c, n _ 0 \in \mathbb{ R } ^ + \text{ s.t. } \forall n \in \mathbb{ N } _ 0, n \ge n _ 0 \implies g \left( n \right) \ge c \cdot f \left( n \right) \]
Omega of
Let \( f, g : \mathbb{ N } _ 0 \to \mathbb{ R } ^ { \ge 0 } \), we say that \( g \) is omega of \( f \) iff \( g \in \Omega \left( f \right) \)

Here we can think of \( \Omega \) as being \( \mathcal{ O } \)'s counter part, so that when \( g \) is omega of \( f \), then \( f \) is a lower bound on \( g \)'s eventual growth rate.

Theta
Suppose we have \( f: \mathbb{N}_{0} \to \mathbb{R}^{\ge 0} \), then we define the set \( \Theta \left( f \right) \) as the set of all functions \( g : \mathbb{N}_{0} \to \mathbb{R}^{\ge 0} \) such that \[ g \in \mathcal{ O } \left( f \right) \cap \Omega \left( f \right) \]

The point of theta is to say that \( g \) is both eventually upper and lower bounded by \( f \) which means that \( g \) should grow asymptotically the same as \( f \)
