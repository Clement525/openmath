Linear Image Filter
A linear image filter is a linear transformation mapping continous functions to continuous functions.
Discrete Convolution
Suppose that \( X \subseteq \mathbb{ N } _1 \) and \( f, g : X \to \mathbb{ R } \) are functions then we define the function \( f \ast g : X \times X \to \mathbb{ R } \) by \[ \left( f \ast g \right) \left( i, j \right) = \sum _ { k, l \in X } f \left( k, l \right) h \left( i - k , j - l \right) \]
Continuous Convolution
Suppose that \( X \subseteq \mathbb{ R } ^ n \) and that \( f, g : X \to \mathbb{ R } \) are functions then we define the function \( f * g \) as \[ \left( f * g \right) \left( x \right) = \int _ { - \infty } ^ \infty f \left( \lambda \right) h \left( x - \lambda \right) d \lambda \]
Linear Shift Invariant Filter
Suppose that a \( l \) is a linear filter, and suppose that we have \( l \left( f \right) = g \) then we say that \( l \) is shift invariant when for any \( p \in \operatorname{ dom } \left( f \right) \) we have \[ l \left( f \left( x - p \right) \right) = g \left( x - p \right) \]
