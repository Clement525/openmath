intersected topology
Let `X` be a topological space with toplogy `cc T`. If `Y` is a subset of `X`, then we define the the toplogy `cc T` intersected with `Y` as the set
`cc T_(nn Y) := {Y nn U : U in cc T}`
intersected topology is a topology
The set `T_(nn Y)` is a topology on `Y`

We'll show that `O/` and `Y` are elements of `T_(nn Y)`, this can be seen as since `O/ in T`, then `Y nn O/ = O/ in T_(nn Y)` and also since `Y sube X`, then `Y nn X = Y` so `Y in T _ (nn Y)`

Now we want to prove that `T _(nn Y)` is closed under arbitrary unions, which we know because `bigcup_(alpha in I) (U_alpha nn Y) = (bigcup_(alpha in I) U_alpha) nn Y`. Similarly we can see that it's closed under finite intersections as `(U_1 nn Y) nn ... nn (U_n nn Y) = (U_1 nn ... nn U_n) nn Y`.

Therefore we know that `T_(nn Y)` is a topology.

subspace topology
Since `T_(nn Y)` forms a topology, we will denote it by `cc T_(nn Y)` and call it the subspace topology and that `Y` is a subspace of `X`
basis for the subspace topology
Suppose that `cc B` is a basis for the topology of `X`, then `M := {Y nn B : B in cc B}` is a basis and it generates `cc T _(nn Y)`

To prove this statement true, we'll use the handy basis criterion

First note that given any element `Y nn B in M`, we know that it is open with respect to `cc T_(nn Y)`, because `B` is open with respect to `X`

Now let `Y nn U` be an open set of `Y` which means that `U` is open in `X`. Let `p in Y nn U` , since `p in U` , `U` is open in `X` and `cc B` is a basis for `X` then we know there is some `B in cc B` such that `p in B sube U` if that's the case then we know that `p in Y nn B sube Y nn U`, noting that `Y nn B in M`, therefore we know that `M` is a basis and it is for `Y`

open in a subspace implies open
Suppose that `Y` is a subspace of `X` and also that `Y` is open in `X`, then any set that's open in `Y` is also open in `X`
Suppose that `U` is open in `Y`, thus there is some `V` open in `X` such that `U = Y nn V`, but since `Y` is open in `X`, then as a finite intersection of open elements, then `U` is also open in `X`, as needed.
