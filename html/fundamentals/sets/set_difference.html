DeMorgan's Laws
Suppose that `M` is a family of sets, then `X \\ (bigcup_(A in M) A) = bigcap_(A in M) (X \\ A)` and `X \\ (bigcap_(A in M) A) = bigcup_(A in M) (X \\ A)`

Note that, `p in X \\ (bigcup_(A in M) A)`, if and only if `p in X` and `x notin bigcup_(A in M) A`, which is equivalent to: for all `A in M` `p notin A`, which is the same as `p in (X \\ A)` for each `A in M`, so by definition `p in bigcap_(A in M) (X \\ A)`. Since each connective in the previous paragraph was an iff, then this implies the two sets are equal

For the second proof, we'll follow a similar structure to the first. Note that, `p in X \\ (bigcap_(A in M) A)`, if and only if `p in X` and `x notin bigcap_(A in M) A`, which is equivalent to: there is some `A in M` `p notin A`, which is the same as `p in (X \\ A)` for some `A in M`, so by definition `p in bigcup_(A in M) (X \\ A)`. Since each connective in the previous paragraph was an iff, then this implies the two sets are equal
