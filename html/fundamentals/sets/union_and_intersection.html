Union and Intersection

union
Given two sets `A, B sube X`, then the union of `A` and `B` is defined as the set `A uu B := {p in X: p in A text( or ) p in B }`
intersection
Given two sets `A, B sube X`, then the intersection of `A` and `B` is defined as the set `A nn B := {p in X: p in A text( and ) p in B }`
arbitrary union
Suppose that `M` is a family of sets, then `bigcup M` is defined so that `X in bigcup M iff EE A in M, X in A`
arbitrary union element of notation
We define `bigcup_(A in M) A :=` `bigcup M`
arbitrary union indexed notation
Suppose that `I` is an index set for the collection `cc A = {A_i : i in I}` where `A_i` is a set, then `bigcup_(i in I) A_i := ` `bigcup cc A`
arbitrary union counting notation
Suppose that `a, b in ZZ`, with `a < b`, then `bigcup_(i = a)^b A_i = ` `bigcup {A_i: i in ZZ, a le i le b}`
disjoint sets
Given two sets `A,B` we say that `A` and `B` are disjoint when `A nn B` = `O/`
pairwise disjoint sets
Suppose that `M` is a family of sets, then we say these sets are pairwise disjoint when given `A, B in M` such that `A ne B`, then `A nn B = O/`
partition
Suppose that `X` is a set, then we say that a set `P` is a partition of `X` if and only if the following are true
  • `O/ notin P`
  • `bigcup P` ` = X`
  • P is pairwise disjoint
partition of the integers
The family `{ {p in ZZ: p < 0}, {0}, {p in ZZ: p > 0}}` is a partition of `ZZ`

Note that the integers `-1, 0, 1` show that the empty set is not in this family. We'll now prove that it's union equals `ZZ`, so let `p in bigcup P`, therefore `p` is in at least one of the sets included in `P`, since each is a subset of `ZZ`, then we know `p in ZZ`, for the reverse direction, we can assume that `p in ZZ`, therefore we know that `p` is either positive, negative or zero, so that `p in bigcup P`, this shows `bigcup P = ZZ`.

To show this family is pairwise disjoint, note that `0 notin {p in ZZ: p < 0}` and `0 notin {p in ZZ: p > 0}`, which shows us that `{0} nn {p in ZZ: p < 0} = O/` and `{0} nn {p in ZZ: p > 0} = O/`.

Given any positive integer, we know it cannot be a negative integer, therefore `{p in ZZ: p < 0} nn {p in ZZ: p> 0} = O/`, thus we know that `P` is pairwise disjoint, which concludes the proof.

a set intersected with a superset is itself
Let `A, B` be sets such that `A sube B`, then `A nn B = A`

We start by showing `A nn B sube A`, so consider `x in A nn B`, so we know that `x in A` and `x in B`, therefore we trivially know that `x in A`, as needed

Now consider the other direction, we need to show that `A sube A nn B`, so consider that `x in A`, then we want to show that `x in A` and `x in B`, which really just simplifies to showing `x in B`, but we know that `A sube B`, therefore since `x in A`, we know that `x in B` finishing the proof

intersection factors from union
Suppose that `U_alpha` is an indexed family of sets, and `Y` is any set, then
`bigcup_(alpha in I)(U_alpha nn Y) = (bigcup_(alpha in I) U_alpha) nn Y`

Suppose that `x in bigcup_(alpha in I)(U_alpha nn Y)`, therefore there is some `beta in I` such that `x in U_beta nn Y`, so that `x in U_beta` and `x in Y`. Since there is some `beta in I` such that `x in U_beta` then `x in bigcup_(alpha in I) U_alpha`, additionally we had that `x in Y` so that `x in (bigcup_(alpha in I) U_alpha) nn Y` as needed

Now suppose that `x in (bigcup_(alpha in I) U_alpha) nn Y`, therefore `x in (bigcup_(alpha in I) U_alpha)` and `x in Y`, due to this we have some `beta in I`, such that `x in U_beta` and thus `x in U_beta nn Y` which by definition shows that `x in bigcup_(alpha in I)(U_alpha nn Y)`

intersection factors from intersection
Suppose that `U_alpha` is an indexed family of sets, and `Y` is any set, then
`bigcap_(alpha in I)(U_alpha nn Y) = (bigcap_(alpha in I) U_alpha) nn Y`

Suppose that `x in bigcap_(alpha in I)(U_alpha nn Y)`, therefore for every `alpha in I` we have: `x in U_alpha nn Y`, so that `x in U_alpha` and `x in Y`. Since it's true for every `alpha in I` then `x in bigcap_(alpha in I) U_alpha`, additionally we had that `x in Y` so that `x in (bigcap_(alpha in I) U_alpha) nn Y` as needed

Now suppose that `x in (bigcap_(alpha in I) U_alpha) nn Y`, therefore `x in (bigcap_(alpha in I) U_alpha)` and `x in Y`, due to this, for every `alpha in I`, we know `x in U_alpha` and thus `x in U_alpha nn Y` which by definition shows that `x in bigcap_(alpha in I)(U_alpha nn Y)`

union of subsets is still a subset
Suppose that `cc C` is a collection of subsets of `X`, then `bigcup_{C in cc C} C sube X`
Let `x in bigcup_{C in cc C} C`, then `x in U` for some `U in cc C`, since `U` must be a subset of `X` then x is also in `X`. As needed
a set covered in subsets is a union
Suppose `A` is a set and that for each `a in A`, there is a `B_a` such that `a in B_a sube A`, then `A = bigcup_{a in A} B_a`

We can see that `cc C = {B_a: a in A}` is a collection of subsets of `A`, so since a union of a subsets is still a subset we have `bigcup_{a in A} B_a sube A`.

But also given `p in A` we know that `p in B_p` so `p in bigcup_{a in A} B_a` which shows `A sube bigcup_{a in A} B_a`, so we can conclude `A = bigcup_{a in A} B_a`.
