Characteristic Function of a Set
Suppose that \( A \) is a set, then it's characteristic function is defined as \[ \chi _ A \left( x \right) = \begin{cases} 1 &, \text{ if } x \in A \\ 0 &, \text{ if } x \notin A \end{cases} \]
Set Equality through the Characteristic
\[ A = B \iff \chi _ A = \chi _ B \]
Symmetric Difference
Suppose \( A, B \) are two sets, then we define \[ A \Delta B := \left( A \setminus B \right) \cup \left( B \setminus A \right) \] using the set difference
Symmetric Difference Characterization
Suppose that \( A, B \) are sets then \[ x \notin A \Delta B \iff \left( x \in A \cap B \lor \left( x \notin A \land x \notin B \right) \right) \]
Characteristic Version of Symmetric Difference
\[ \chi _ { A \Delta B } = \chi _ A \oplus \chi _ B \]
Symmetric Difference is Associative
For any three sets \( A, B, C \) we have \[ \left( A \Delta B \right) \Delta C = A \Delta \left( B \Delta C \right) \]
\[ \begin{align} \chi _ { \left( A \Delta B \right) \Delta C } &= \left( \chi _ A \oplus \chi _ B \right) \oplus \chi _ C \\ &= \chi _ A \oplus \left( \chi _ B \oplus \chi _ C \right) \\ &= \chi _ { A \Delta \left( B \Delta C \right) } \end{align} \] therefore \( \left( A \Delta B \right) \Delta C = A \Delta \left( B \Delta C \right)\)
Characteristic Version of Symmetric Difference Generalized
Let \( n \in \mathbb{ N } _ 2 \), prove that for any collection of sets \( A_1 , A_2 , \ldots , A_n\) we have that \[ \chi _ {A_1 \Delta A_2 \Delta \cdots \Delta A_n } = \chi _ { A _ 1 } \oplus \chi _ { A _ 2 } \oplus \cdots \oplus \chi _ { A _ n } \]
Holds by induction using previous facts to help with the induction step.
