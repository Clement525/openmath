Summation
Given a sequence \( \left ( a_{n} \right ) \), then we define \[ \sum_{i = 1}^{n} a _ i := a_{1} + a_{2} + \ldots + a_{n} \]
Constant can be Factored
Suppose \( c \in \mathbb{R} \), then \( \sum_{i = 1}^{n} c a_{i} = c \sum_{i = 1}^{n} a_{i} \)

Sum of Consecutive Integers
