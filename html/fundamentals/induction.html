induction
Suppose that \( a \in \mathbb{Z} \) and \( P \left ( k \right ) : \mathbb{Z}^{\ge a} \to \left \lbrace \mathbf{T} , \mathbf{F} \right \rbrace \) is a predicate, if \( P \left ( a \right ) \) holds true and \( P \left ( k \right ) \implies P \left ( k + 1 \right ) \), then \( P \left ( k \right ) \) holds true for all \( k \in \mathbb{Z}^{\ge a} \)
Let \( k \in \mathbb{Z}^{\ge a} \), if \( k = a \), then we know that \( P \left ( k \right ) \) is true, otherwise if \( k \gt a \), then \( k = a + r \) for some \( r \in \mathbb{N}_{1} \), therefore \( P \left ( a \right ) \implies P \left ( a + 1 \right ) \), so we know that \( P \left ( a + 1 \right ) \) is true, then applying the same reasonging \( P \left ( a + 1 \right ) \implies P \left ( a + 2 \right ) \) so \( P \left ( a + 2 \right ) \) is true, thus we can see that after \( r \) iterations of that \( P \left ( a + r \right ) = P \left ( k \right ) \) is also true.
linear recursive sum
Suppose that \( \left ( x_{n} \right )_{n \in \mathbb{N}_{0}} \) is a sequence such that for any \( k \in \mathbb{N}_{1} \) we have \( x_{k} := a x_{k - 1} + b \), where \( a , b \in \mathbb{R} \), prove that
\( x_{k} = a^{k} \left ( x_{0} - \frac{b}{1 - a} \right ) + \frac{b}{1 - a} \)

We proceed with a proof by induction so let \( k \in \mathbb{N}_{1} \), suppose that \( k = 1 \), thus by the definition of \( x_{1} \) we know it's equal to \( a \cdot x_{0} + b \), now some algebra: \( a \cdot x_{0} + b = a \cdot x_{0} + b \left ( \frac{1 - a}{1 - a} \right ) = a^{1} \cdot x_{0} - \frac{b a}{1 - a} + \frac{b}{1 - a} = a^{k} \left ( x_{0} - \frac{b}{1 - a} \right ) + \frac{b}{1 - a} \), which is what we needed to show.

Now for the induction step, so suppose that \( k \gt 1 \) and that \( x_{k} = a^{k} \left ( x_{0} - \frac{b}{1 - a} \right ) + \frac{b}{1 - a} \) holds true (and call it our Induction Hypothesis), we'll now show that \( x_{k + 1} = a^{k + 1} \left ( x_{0} - \frac{b}{1 - a} \right ) + \frac{b}{1 - a} \).

First recall the very definition of \( x_{k + 1} \) which is that \( x_{k + 1} = a x_{k} + b \), but from our induction hypothesis, we know that \( x_{k} = a^{k} \left ( x_{0} - \frac{b}{1 - a} \right ) + \frac{b}{1 - a} \), so therefore \( x_{k + 1} = a \left [ a^{k} \left ( x_{0} - \frac{b}{1 - a} \right ) + \frac{b}{1 - a} \right ] + b \).

We'll start by multiplying everything out and then seeing what we get: \( a \left [ a^{k} \left ( x_{0} - \frac{b}{1 - a} \right ) + \frac{b}{1 - a} \right ] + b = a^{k + 1} \left ( x_{0} - \frac{b}{1 - a} \right ) + \frac{a b}{1 - a} + b = a^{k + 1} \left ( x_{0} - \frac{b}{1 - a} \right ) + \frac{a b}{1 - a} + b \left ( \frac{1 - a}{1 - a} \right ) \) \( = a^{k + 1} \left ( x_{0} - \frac{b}{1 - a} \right ) + \frac{a b + b - a b}{1 - a} \) \( = a^{k + 1} \left ( x_{0} - \frac{b}{1 - a} \right ) + \frac{b}{1 - a} \) which is exactly what we set out to prove.

Therefore since the statement holds for \( k = 1 \), and if when it holds for \( k \) then it also holds for \( k + 1 \), then the statement is true for all \( k \in \mathbb{N}_{1} \), as needed.
