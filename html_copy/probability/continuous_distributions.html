Continuous Random Variable
We say that a random variable \( X \) is continuous if \[ P \left( X = x \right) = 0 \] for every \( x \in \mathbb{ R } \)
Continuous Uniform Distribution
The random variable \(U\) is said to have a (continuous) uniform distribution on the unit interval \([0,1]\) denoted by \( U \sim \text { unif }[0,1]\) iff \[ P(U \leq u)=u \quad \forall 0 \leq u \leq 1 . \]
Probability of an Interval in the Uniform Distribution
\[ P \left( a \le X \le b \right) = b - a \]
Density Function
Let \( f: \mathbb{ R } \to \mathbb{ R } \) be a function, then we say that \( f \) is a density function if \( f \left( x \right) \ge 0 \) for any \( x \in \mathbb{ R } \) and \[ \int _ {-\infty} ^ \infty f \left( x \right) dx = 1 \]
Absolutely Continuous Random Variable
A random varaible \( X \) is said to be absolutely continuousif there is a density function \( f \) such that \[ P \left( a \le X \le b \right) = \int _ a ^ b f \left( x \right) dx \] whenever \( a \le b \in \mathbb{ R } \)
A Random Variable has a Density Function
We say that a random variable \( X \) has a density function \( f \) if it is absolutely continuous using this density function.
Probability of a Point is Zero for an Absolutely Continuous Random Variable
Suppose that \( X \) is an absolutely continuous random variable then \[ P \left( X = a \right) = 0 \] for any \( a \in \mathbb{ R } \)
Exponential 1 Distribution
Suppose that \( Z \) is a random varaible, then we write \( Z \sim \operatorname{ exp } \left( 1 \right) \) iff \[ Z = - \ln \left( U \right) \] where \( U \sim \operatorname{ unif } \left[ 0, 1 \right] \)
Probability of an Interval in the Exponential Distribution
Suppose that \( Z \sim \operatorname{ exp } \left( 1 \right) \), then \[ P \left( s \le Z \le t \right) = e ^ { -s } - e ^ { -t } \]
Observe that the following sets are equal: \[ \begin{align} s \le Z \le t &\boldsymbol{=} s \le -\ln \left( U \right) \le t \\ &\boldsymbol{=} -t \le \ln \left( U \right) \le -s \\ &\boldsymbol{=} e ^ { -t } \le U \le e ^ { -s } \\ \end{align} \] therefore we know that \[ P \left( s \le Z \le t \right) = P \left( e ^ { -t } \le U \le e ^ { -s } \right) = e ^ { -s } - e ^ { -t } \]
