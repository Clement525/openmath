openmath editor

Welcome to the openmath editor. Up until now the only way people could edit openmath to do major edits was by cloning the files to the computer, modifying them and then making a pull request. This editor was created to reduce the barrier of entry to making changes on openmath.

fast insertion menu

Simply press shift+I to toggle it, and search for the latex command you want to enter, use tab to select it and then enter to insert it into the document at the last position you were at.

editing

To modify the files simply change anything that already exists in the left source code panel.

To create a new definition go to the Copy Element dropdown, and select definition. This copies a blank definition to your clipboard which you can paste into the editor panel.

Likewise to create a theorem, proposition, lemma, corollary or exercise you can use the copy dropdown to select the correct one.

One of the most powerful tools on openmath is the knowledge link, which allows you to link to other places on the site and it can be expanded in-line. Simply copy the knowledge link from the copy menu, then navigate to the location of the knowledge you'd like to link to and use that as the href tag.

request changes

After you've made your edits, click the request changes file. This immediately copies all the text in the left input box to your clipboard and opens the respective page you just edited on github. Next simply paste the contents of your clipboard to overwrite the previous content on the page and follow the steps given to you by github.
