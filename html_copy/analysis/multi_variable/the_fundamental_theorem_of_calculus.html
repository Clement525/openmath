Fundamental Theorem of Calculus I
Let \( f \) be integrable on \( \left[ a, b \right] \) and define \( F \left( x \right) = \int _ { a } ^ { x } f \left( t \right) dt \) for \( x \in \left[ a, b \right] \). Then \( F \) is continuous, moreover when \( f \) is continuous at \( x _ 0 \in \left( a, b \right) \) then we have that \( F \) is differentiable at \( x _ 0 \) and \[ F ^ \prime \left( x _ 0 \right) = f \left( x _ 0 \right) \]
Fundamental Theorem of Calculus II
Let \( f \) be integrable on \( \left[ a, b \right] \), if there is a continuous function \( F: \left[ a, b \right] \to \mathbb{ R } \) that is differentiable on \( \left( a, b \right) \) such that \( F ^ \prime \left( x \right) = f \left( x \right) \) for every \( x \in \left( a, b \right) \) then \[ \int _ { a } ^ { b } f \left( x \right) dx = F \left( b \right) - F \left( a \right) \]
MVT for Integrals
Suppose that \( f: \left[ a, b \right] \to \mathbb{ R } \) is continous, then there exists a point \( c \in \left( a, b \right) \) such that \[ \frac{1}{b - a} \int _ a ^ b f \left( x \right) dx = f \left( c \right) \]

Let \( y \in \left[ a, b \right] \) and define \( F \left( t \right) = \int _ y ^ t \) for any \( t \in \left[ a, b \right] \), since \( f \) is continuous on \( \left[ a, b \right] \) that means that for every \( x _ 0 \in \left( a, b \right) \) we have that \( F ^ \prime \left( x _ 0 \right) = f \left( x _ 0 \right)\), and also that \( F \) is continuous on \( \left[ a, b \right] \).

As noted above we satisfy the conditions to the mean value theorem which means that there is some \( c \in \left( a, b \right) \) such that \[ \frac{F \left( b \right) - F \left( a \right) }{ b - a } = F ^ \prime \left( c \right) = f \left( c \right) \]

Finally we note that \[ \begin{align} F \left( b \right) - F \left( a \right) &= \int _ y ^ b f \left( x \right) dx - \int _ y ^ a f \left( x \right) dx \\ &= \int _ y ^ b f \left( x \right) dx + \int _ a ^ y f \left( x \right) dx \\ &= \int _ a ^ b f \left( x \right) dx \\ \end{align} \]

Which shows that \[ \frac{1}{b - a} \int _ a ^ b f \left( x \right) dx = f \left( c \right) \]

Antiderivative
A function \( F \) is an antiderivative of the function \( f \) if \[ F ^ \prime \left( x \right) = f \left( x \right) \] for every \( x \in \operatorname{ dom } \left( f \right) \)
Sign Function
We define \( \operatorname{ sign } : \mathbb{ R } \to \left\{ -1, 0, 1 \right\} \) such that \[ \operatorname{ sign } \left( x \right) = \begin{cases} 1 &\text{ if } x \gt 0\\ 0 &\text{ if } x = 0 \\ -1 &\text{ if } x \lt 0 \end{cases} \]
Sign Functions Integral
Let \( f \left( x \right) = \operatorname{ sign } \left( x \right) \) and \( F \left( x \right) = \left\lvert x \right\rvert \), show that \( f \) is riemann integrable on \( \left[ a, b \right] \) and that \( \int _ a ^ b f \left( x \right) dx = F \left( b \right) - F \left( a \right) \) for any \( a \lt b \), why is \( F \) not an antiderivative of \( f \).
Note that if \( 0 \lt a \lt b \) then \[ \int _ a ^ b \operatorname{ sign } \left( x \right) dx = \int _ a ^ b 1 dx = b - a = \left\lvert b \right\rvert - \left\lvert \right\rvert \] when \( 0 \gt b \gt a \) then we have \[ \int _ a ^ b \operatorname{ sign } \left( x \right) dx = \int _ a ^ b -1 dx = a - b = = -b - \left( -a \right) = \left\lvert b \right\rvert - \left\lvert a \right\rvert \] and finally when \( a \lt 0 \lt b \) then we have \[ \int _ a ^ b \operatorname{ sign } \left( x \right) dx = \int _ a ^ 0 -1 dx + \int _ 0 ^ b 1 dx = a + b = b - \left( -a \right) = \left\lvert b \right\rvert - \left\lvert a \right\rvert \] which shows that \( \int _ a ^ b f \left( x \right) dx = F \left( b \right) - F \left( a \right) \) in all cases, note that \( F \) is not an antiderivative of \( f \) because it is undefined at some points in the domain of \( f \).
One over Epsilon Integral
Suppose that \( f: \mathbb{ R } \to \mathbb{ R } \) is continuous and fix \( \epsilon \in \mathbb{ R } ^ + \), define \( G \left( x \right) = \frac{1}{\epsilon } \int _ x ^ { x + \epsilon } f \left( t \right) dt \), show that \( G \) is \( C ^ 1 \) and compute \( G ^ \prime \).
Let's try to compute \( G ^ \prime \), if it turns out that by observation that equation we get for \( G ^ \prime \) is continous then \( G \) is \( C ^ 1 \), we approach by definition: \[ G ^ \prime \left( x \right) = \lim _ { h \to 0 } \frac{\int _ { x + h } ^ { x + h + \epsilon } f \left( t \right) dt - \int _ { x } ^ { x + \epsilon } f \left( t \right) dt }{h} \] Since \( h \to 0 \) we can assume that eventually \( h \lt \epsilon \) so that \[ x \lt x + h \lt x + \epsilon \lt x + h + \epsilon \] therefore the integrals have an overlapping cancellation from \( x + h \) to \( x + \epsilon \) so that \[ \epsilon \cdot G ^ \prime \left( x \right) = \lim _ { h \to \infty } \frac{\int _ { x + \epsilon } ^ { x + h + \epsilon } f \left( t \right) dt - \int _ { x } ^ { x + h } f \left( t \right) dt }{h} \] Let \( a \in \mathbb{ R } \) and define \( K \left( b \right) = \int _ a ^ b f \left( t \right) dt \) so that we have \[ \int _ { x + \epsilon } ^ { x + h + \epsilon } f \left( t \right) dt = - \int _ { a } ^ { x + \epsilon } f \left( t \right) dt + \int _ { a } ^ { x + h + \epsilon } f \left( t \right) dt = K \left( x + h + \epsilon \right) - K \left( x + \epsilon \right) \] and also \[ \int _ { x } ^ { x + h } = - \int _ { a } ^ { x } f \left( t \right) dt + \int _ { a } ^ { x + h } f \left( t \right) dt = K \left( x + h \right) - K \left( x \right) \] Therefore by finally constructing \( J \left( p \right) = K \left( p + \epsilon \right) - K \left( p \right) \) we have \[ \begin{align} \epsilon \cdot G ^ \prime \left( x \right) &= \lim _ { h \to \infty } \frac{\int _ { x + \epsilon } ^ { x + h + \epsilon } f \left( t \right) dt - \int _ { x } ^ { x + h } f \left( t \right) dt }{h}\\ &= \lim _ { h \to \infty } \frac{\left( K \left( x + h + \epsilon \right) - \left( K \left( x + \epsilon \right) - K \left( x \right) \right) }{ h} \\ &= \lim _ { h \to \infty } \frac{J \left( x + h \right) - J \left( x \right) }{h} \\ &= J ^ \prime \left( x \right) \\ &= K ^ \prime \left( x + \epsilon \right) - K ^ \prime \left( x \right) \\ &= f \left( x + \epsilon \right) - f \left( x \right) \\ \end{align} \] therefore \[ G ^ \prime \left( x \right) = \frac{f \left( x + \epsilon \right) - f \left( x \right) }{\epsilon } \] as the right hand side is composition of fundamental operations on \( \mathbb{ R } \) which maintain continuity then \( G ^ \prime \left( x \right) \) is continuous, so that \( G \) is \( C ^ 1 \) as needed.
Sup Norm Sandwich Inequality
Suppose that \( f: \mathbb{ R } \to \mathbb{ R } \) such that \( S = \lVert f \rVert _ \infty \) then for any \( x \in \mathbb{ R } \) we have \[ -S \le f \left( x \right) \le S \]
Let \( x \in \mathbb{ R } \) then we have \[ f \left( x \right) \le \left\lvert f \left( x \right) \right\rvert \le \lVert f \rVert _ \infty = S \] therefore \[ -S \le f \left( x \right) \le S \]
Twice Differentiable Triangle
Suppose that \( f \) is twice differentiable on \( \mathbb{ R } \), \( \lVert f \rVert _ \infty = A \) and that \( \lVert f ^ { \prime \prime } \rVert _ \infty = C \). Prove that \( \lVert f ^ \prime \rVert _ \infty \le \sqrt{ 2 A C } \)
Let \( x _ 0 \in \mathbb{ R } \), and let \( f ^ \prime \left( x _ 0 \right) = b \), if it's the case that \( b \gt 0 \), then let \( t \in \mathbb{ R } \) if \( t \gt 0 \) then we apply MVT on the interval \( \left[ x _ 0, x _ 0 + t \right] \) to get some \( \theta _ 1 \in \left( x _ 0, x _ 0 + t \right) \) such that \[ \frac{f ^ \prime \left( x _ 0 + t \right) - f ^ \prime \left( x _ 0 \right) }{ t} = f ^ { \prime \prime } \left( \theta _ 1 \right) \] so that \[ f ^ \prime \left( x _ 0 + t \right) = f ^ { \prime \prime } \left( \theta _ 1 \right) t + b \] and that \[ f ^ \prime \left( x _ 0 + t \right) \ge b - C t = b - C \left\lvert t \right\rvert \] Now if \( t \lt 0 \) one can do the same analysis but will obtain some \( \theta _ 2 \in \left[ x _ 0 + t, x _ 0 \right] \) such that \[ \frac{f ^ \prime \left( x _ 0 \right) - f ^ \prime \left( x _ 0 + t \right) }{-t} = f ^ { \prime \prime } \left( \theta _ 2 \right) \] so that \[ f ^ \prime \left( x _ 0 + t \right) = f ^ \prime \left( x _ 0 \right) - f ^ { \prime \prime } \left( \theta _ 2 \right) \left( -t \right) = b - f ^ { \prime \prime } \left( \theta _ 2 \right) \left\lvert t \right\rvert \ge b - C \left\lvert t \right\rvert \] Now when \( x _ 0 = 0 \), we have the following integral inequalities, note the the first integral exists because \( f ^ \prime \) is continuous: \[ \int _ { -\frac{b}{C} } ^ { \frac{b}{C} } f ^ \prime \left( t \right) dt \ge \int _ { -\frac{b}{C} } ^ { \frac{b}{C} } b - C \left\lvert t \right\rvert dt \] But note that the integral on the right measure the area of the function \( b - C \left\lvert t \right\rvert \) which hits the y axis at \( b \) and has legs coming down which intersect the \( x \) axis at \( - \frac{b}{C} , \frac{b}{C} \) respectively, so that the total area is given by \( \left( \frac{b}{C} \right) b = \frac{b ^ 2}{C} \). Now focusing on the left hand side we recall that since \( f \) is a continuous function such that \( f ^ \prime \left( x \right) = f ^ \prime \left( x \right) \) then we have: \[ \begin{align} \int _ { -\frac{b}{C} } ^ { \frac{b}{C} } f ^ \prime \left( t \right) dt = f \left( \frac{b}{C} \right) - f \left( -\frac{b}{C} \right) \le 2A \end{align} \] thus chaining previous inqualities with the above equality we have: \[ b \le \sqrt{ 2AC } \] If it turns out that \( b \) is negative we can get the same bound, which shows for an arbitrary choice of \( x _ 0 \) we were able to deduce that \( \left\lvert f \left( x _ 0 \right) \right\rvert \le \sqrt{ 2AC } \), thus \[ \lVert f ^ \prime \rVert _ \infty \le \sqrt{ 2AC } \]
