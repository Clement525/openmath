Intermediate Value
Suppose \( f: \left[ a, b \right] \to \mathbb{ R } \) is continuous and \( z \in \mathbb{ R } \) such that \( f \left( a \right) \lt z \lt f \left( b \right) \) then there exists a point \( c \in \left( a, b \right) \) such that \[ f \left( c \right) = z \]
Path
A path in \( S \subseteq \mathbb{ R } ^ n \) from \( a \) to \( b \) (elements of \( S \)) is the image of a continuous function \( \gamma : \left[ 0, 1 \right] \to S \) such that \( \gamma \left( 0 \right) = a \) and \( \gamma \left( 1 \right) = b \)
Intermediate Value for Paths
Suppose that \( S \subseteq \mathbb{ R } ^ n \) and \( f \) is a continuous real-valued function on \( S \). If there is a path from \( a \) to \( b \) in \( S \) and \( z \in \mathbb{ R } \) such that \( f \left( a \right) \lt z \lt f \left( b \right) \) then there is a point \( c \) on the path such that \( f \left( c \right) = z \)

Select 3 distinct points on the circle, \( x, y, z \in C \subseteq \mathbb{ R } ^ 2 \), if any of \( f \left( x \right) , f \left( y \right), f \left( z \right) \) were equal than we would be done, so assume they are distinct, and without loss of generality that \( f \left( x \right) \lt f \left( y \right) \lt f \left( y \right) \).

We can make the Absolute value of a Polynomial as Small as we Want
Let \( \epsilon \in \mathbb{ R } ^ + \) then there exists a \( \delta \in \mathbb{ R } ^ + \) such that \[ \left\lvert x \right\rvert \lt \delta \implies \left\lvert p \left( x \right) \right\rvert \lt \epsilon \]
Suppose that \( p \left( x \right) = \sum _ { i = 0 } ^ { n } a _ i x ^ i \) we note that \[ \left\lvert p \left( x \right) \right\rvert \le \sum _ { i = 0 } ^ { n } \left\lvert a _ i \right\rvert \left\lvert x ^ i \right\rvert \] Therefore by selecting \[ \delta = \min \left( 1, \frac{\epsilon }{ \left( n + 1 \right) \cdot \max \left\{ \left\lvert a _ i \right\rvert : i \in \left[ 1, \ldots , n \right] \right \} } \right) \] choosing the min of one in the above so that we have the property that for any \( i \in \mathbb{ N } _ 0 \) we we have \( \left\lvert x \right\rvert ^ i \le \left\lvert x \right\rvert \) showing us that \[ \begin{align} \left\lvert p \left( x \right) \right\rvert &\le \sum _ { i = 0 } ^ { n } \left\lvert a _ i \right\rvert \left\lvert x ^ i \right\rvert \\ &\le \sum _ { i = 0 } ^ { n } \left\lvert a _ i \right\rvert \left\lvert x \right\rvert \\ &\le \sum _ { i = 0 } ^ { n } \frac{\left\lvert a _ i \right\rvert}{\max \left\{ \left\lvert a _ i \right\rvert : i \in \left[ 1, \ldots , n \right] \right \}} \left( \frac{\epsilon }{ n + 1 } \right)\\ &\le \sum _ { i = 0 } ^ { n } 1 \left( \frac{\epsilon }{n + 1 } \right) \\ &\le \epsilon \end{align} \]
We can make the Absolute value of a Polynomial as small as we Want Inverse Edition
Let \( \epsilon \in \mathbb{ R } ^ + \) then there exists a \( \delta \in \mathbb{ R } ^ + \) such that \[ \left\lvert x \right\rvert \gt \delta \implies \left\lvert p \left( x ^ { -1 } \right) \right\rvert \lt \epsilon \]
Let \( \epsilon \in \mathbb{ R } ^ + \) by the above we obtain some \( \delta ^ \prime \in \mathbb{ R } ^ + \) such that if \( \left\lvert x \right\rvert \lt \delta ^ \prime \) then \( \left\lvert p \left( x \right) \right\rvert \lt \epsilon \), take \( \delta = \frac{1}{\delta ^ \prime} \) and assume that \( \left\lvert x \right\rvert \gt \frac{1}{\delta ^ \prime} \) therefore \( \delta ^ \prime \gt \frac{1}{\left\lvert x \right\rvert } = \left\lvert \frac{1}{x} \right\rvert \) therefore we have that \( \left\lvert p \left( x ^ { -1 } \right) \right\rvert \lt \epsilon \) as needed.
A Polynomial of Odd Degree has at Least One Root
Any monic polynomial \( p \left( x \right) \in \mathbb{ R } \left[ x \right] \) such that \( \operatorname{ deg } \left( p \right) = 2k + 1 \) for some \( k \in \mathbb{ N } _ 0 \) has at least one real root.
We approach with the intermediate value theorem in mind, we note that \[ \frac{p \left( x \right) }{x ^ { 2k + 1 } } = 1 + \sum _ { n = 0 } ^ { 2k } a _ n x ^ { n - \left( 2k + 1 \right) } \] and that \( n - \left( 2k + 1 \right) \le -1 \) therefore let \( q \left( x \right) = \sum _ { n = 0 } ^ { 2k } a _ n x ^ { 2k + 1 - n } \) so that we have \[ \frac{p \left( x \right) }{x ^ { 2k + 1 } } = 1 + q \left( x ^ { -1 } \right) \] Now note that for any \( \epsilon \in \left( 0, 1 \right) \) there is some \( \delta \in \mathbb{ R } ^ + \) such that if \( \left\lvert x \right\rvert \gt \delta \) we have \( \left\lvert q \left( x ^ { -1 } \right) \right\rvert \lt \epsilon \), so that \( q \left( x ^ { -1 } \right) \ge -\epsilon \) to allow us to say: \[ \frac{p \left( x \right) }{x ^ { 2k + 1 } } = 1 + q \left( x ^ { -1 } \right) \gt 1 - \epsilon \] so that we conclude \( p \left( x \right) \gt x ^ { 2k + 1 } - \epsilon \cdot x ^ { 2k + 1 } = \left( 1 - \epsilon \right) x ^ { 2k + 1 } \) noticing that \( 1 - \epsilon \in \mathbb{ R } ^ + \) thus by taking \( a = \delta + 1 \) (so that \( \left\lvert x \right\rvert \gt \delta \) we see that \( p \left( a \right) \gt \left( 1 - \epsilon \right) a ^ { 2k + 1 } \gt 0 \). We also take a moment to note that for symmetrical reasons \( p \left( -a \right) \lt 0\) therefore since \( p \left( -a \right) \lt 0 \lt p \left( 0 \right) \) there exists a point \( c \in \left( -a, a \right) \) such that \( p \left( c \right) = 0 \), as needed.
