Rational Sequence
A rational sequence \( \left( a _ n \right) \) is a sequence such that \( a _ n \in \mathbb{ Q } \)
Cauchy Sequence
Let \( \left( a _ n \right) \) be a rational sequence. We say that \( \left( a _ n \right) \) is a cauchy sequence if for any \( \epsilon \in \mathbb{ R } ^ + \) there exists some \( N \in \mathbb{ N } _ 0 \) such that for every \( n, m \ge N \) \[ \lvert a _ n - a _ m \rvert \le \epsilon \]
A Convergent Rational Sequence is Cauchy
Suppose that \( a _ n \to L \) for some \( L \in \mathbb{ R } \), then \( \left( a _ n \right) \) is cauchy
Let \( \epsilon \in \mathbb{ R } ^ + \) since \( \left( a _ n \right) \) converges to \( q \) then we know that there exists some \( M \in \mathbb{ N } _ 0 \) such that for any \( n \ge N \) we have that \( \lvert a _ n - L \rvert \lt \frac{\epsilon }{2} \), take \( N = M \) and let \( n, m \ge M \) then we have \[ \lvert a _ n - a _ m \rvert = \lvert \left( a _ n - L \right) - \left( a _ m - L \right) \rvert \le \lvert a _ n - L \rvert + \lvert a _ m - q \rvert \lt \frac{\epsilon }{2} + \frac{\epsilon }{2} = \epsilon \] where we've used the triangle inequality
A Cauchy Sequence is Bounded
If \( \left( a _ n \right) \) is a cauchy sequence, then there exists some \( M \in \mathbb{ Q } \) such that \( \lvert a _ n \rvert \le M \)

Since \( \left( a _ n \right) \) is cauchy then with \( \epsilon = 1 \) we get an \( N \in \mathbb{ N } _ 0 \) such that for any \( m, n \ge N \) we have that \( \lvert a _ n - a _ m \rvert \lt 1 \). Specificially since \( N + 1 \ge N \) then we would know that \( \lvert a _ { N + 1 } - a _ m \rvert \le 1 \) which is the same as \[ a _ { N + 1 } -1 \lt a _ n \lt a _ { N + 1 } + 1 \] so that \( \lvert a _ n \rvert \le \max \left( \lvert a _ { N + 1 } - 1 \rvert, \lvert a _ { N + 1 } + 1 \rvert \right) \)

Therefore set \[ M = \max \left( \lvert a _ 0 \rvert, \lvert a _ 1 \rvert, \ldots , \lvert a _ N \rvert, \lvert a _ { N + 1 } - 1 \rvert, \lvert a _ { N + 1 } + 1 \rvert \right) \] With this, let \( k \in \mathbb{ N } _ 0 \) then we know that if \( k \le N \) then \( \lvert a _ k \rvert \le M \) as they are directly included in our maximum, on the other hand if \( k \gt N \) then \( \lvert a _ k \rvert \lt \max \left( \lvert a _ { N + 1 } - 1 \rvert, \lvert a _ { N + 1 } + 1 \rvert \right) \le M \) and therefore \( \left( a _ n \right) \) is bounded.

The Collection of All Cauchy Sequence
We use the notation \( \mathcal{ C } _ \mathbb{ Q } \) to denote the set of all cauchy sequences of rational numbers
Cauchy Subtraction Relation
Let \( \left( a _ n \right), \left( b _ n \right) \in \mathcal{ C } _ \mathbb{ Q } \), and we define the relation such that they are related if \( a _ n - b _ n \to 0 \)
The Cauchy Subtraction Relation is an Equivalence Relation
The cauchy subtraction relation is an equivalence relation

Let \( \left( a _ n \right), \left( b _ n \right), \left( c _ n \right) \in \mathcal{ C } _ \mathbb{ Q } \), let's show that the relation is reflexive, first we have to show that \( a _ n - a _ n \to 0 \) the sequence \( a _ n - a _ n \) is zero for every index so it trivially tends to 0.

To show that the relation is symmetric we assume that \( a _ n - b _ n \to 0 \), and we must show that \( b _ n - a _ n \to 0 \), so let \( \epsilon \in \mathbb{ R } ^ + \), so we get some \( N \in \mathbb{ N } _ 0 \) such that for all \( n \ge \mathbb{ N } _ 0 \) we have that \( \lvert a _ n - b _ n \rvert \lt \epsilon \) but since we can move things around inside of the absolute value bars we get \( \lvert b _ n - a _ n \rvert \lt \epsilon \) as needed.
