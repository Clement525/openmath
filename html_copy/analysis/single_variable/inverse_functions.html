increasing function
given \( A \subseteq \mathbb{R} \), a function \( f{:} A \to \mathbb{R} \) is increasing if given \( x , y \in A \) with \( x \lt y \) we have \( f{\left ( x \right )} \le f{\left ( y \right )} \). An increasing function is also known as a non-decreasing function.
strictly increasing function
given \( A \subseteq \mathbb{R} \), a function \( f{:} A \to \mathbb{R} \) is strictly increasing if given \( x , y \in A \) with \( x \lt y \) we have \( f{\left ( x \right )} \lt f{\left ( y \right )} \).
decreasing function
given \( A \subseteq \mathbb{R} \), a function \( f{:} A \to \mathbb{R} \) is decreasing if given \( x , y \in A \) with \( x \lt y \) we have \( f{\left ( x \right )} \ge f{\left ( y \right )} \). A decreasing function is also known as a non-increasing function.
strictly decreasing function
given \( A \subseteq \mathbb{R} \), a function \( f{:} A \to \mathbb{R} \) is decreasing if given \( x , y \in A \) with \( x \lt y \) we have \( f{\left ( x \right )} \gt f{\left ( y \right )} \)
monotone function
a function defined on a subset of \( \mathbb{R} \) is said to be monotone if and only if it is non-increasing or non-decreasing
strictly monotone function
a function defined on a subset of \( \mathbb{R} \) is said to be monotone if and only if it is stictly increasing or strictly decreasing
invertible function
Let \( f{:} X \to Y \) be a function, if there is a function \( g{:} Y \to X \) such that \( g{\left ( f{\left ( x \right )} \right )} = x \) for all \( x \in X \) and \( f{\left ( g{\left ( y \right )} \right )} = y \) for all \( y \in Y \)
